## 1단계: 기초 데이터 수집 및 변환

### 1.1 입력 정보

| 항목 | 필요 데이터 | 비고 |
|:---|:---|:---|
| 생년월일 | 양력 기준 | 음력 입력 시 양력 변환 필요 |
| 출생 시간 | 시:분 | 모를 경우 시주 제외 또는 추정 |
| 성별 | 남/여 | 대운 순행/역행 판단에 필요 |
| 출생 지역 | 선택사항 | 진태양시 보정에 필요 |

### 1.2 진태양시 보정 (선택)

| 단계 | 계산 방법 |
|:---|:---|
| 표준시 확인 | 한국: 동경 135도 기준 (UTC+9) |
| 실제 경도 확인 | 서울 약 127도 |
| 경도 차이 계산 | 135 - 127 = 8도 |
| 시간 보정 | 8도 × 4분 = 32분 감산 |
| 진태양시 | 출생시간 - 32분 |

### 1.3 만세력 변환

양력 날짜를 사주 간지로 변환합니다.

| 주(柱) | 변환 방법 |
|:---|:---|
| 년주 | 만세력 조회 (입춘 기준 연도 변경) |
| 월주 | 만세력 조회 (절기 기준 월 변경) |
| 일주 | 만세력 조회 (일진표) |
| 시주 | 일간 + 출생 시간으로 계산 |

| 05:00~07:00 | 묘(卯) |
| 07:00~09:00 | 진(辰) |
| 09:00~11:00 | 사(巳) |

## 3단계: 일간 강약 판단

### 3.1 월령(月令) 득령 여부

**월령이란?**
- 월령 = 태어난 달의 지지(地支)
- 사주에서 계절을 나타내며, 만물의 흥망성쇠를 결정하는 가장 강력한 힘

| 월령 | 계절 | 왕성한 오행 |
|:---:|:---:|:---:|
| 인(寅), 묘(卯), 진(辰) | 봄 | 木 |
| 사(巳), 오(午), 미(未) | 여름 | 火 |
| 신(申), 유(酉), 술(戌) | 가을 | 金 |
| 해(亥), 자(子), 축(丑) | 겨울 | 水 |

| 판단 | 조건 | 점수 |
|:---|:---|:---:|
| 득령 | 월지가 일간을 생하거나 같은 오행 | +2 |
| 실령 | 월지가 일간을 극하거나 설기 | -2 |
| 중립 | 그 외 | 0 |

### 3.2 통근(通根) 여부

지지에 일간과 같은 오행 또는 일간을 생하는 오행이 있는지 확인합니다.

| 위치 | 점수 |
|:---|:---:|
| 일지에 뿌리 | +2 |
| 년지/월지/시지에 뿌리 | 각 +1 |

### 3.3 천간 생조 여부

| 조건 | 점수 |
|:---|:---:|
| 인성이 천간에 있음 | +1 |
| 비겁이 천간에 있음 | +1 |

### 3.4 신강/신약 판정

| 총점 | 판정 |
|:---:|:---|
| +3 이상 | 신강(身强) |
| -3 이하 | 신약(身弱) |
| -2 ~ +2 | 중화(中和) 또는 상황에 따라 판단 |

|:---|:---|:---|

| 신약 | 생조(生助) | 인성, 비겁 |
| 중화 | 조후(調候) 우선 | 계절에 따라 다름 |

| 일간 | 여름생 | 겨울생 |
|:---|:---|:---|
| 火 일간 | 水 필요 (식히기) | 木 필요 (살리기) |
| 水 일간 | 金 필요 (근원) | 火 필요 (온기) |
| 土 일간 | 水 필요 (윤택) | 火 필요 (온기) |
| 金 일간 | 水 필요 (씻기) | 火 필요 (제련) |
| 木 일간 | 水 필요 (수분) | 火 필요 (온기) |

### 4.3 용신/희신/기신 정리

| 구분 | 정의 |
|:---|:---|
| 용신(用神) | 사주 균형에 가장 필요한 오행 |
| 희신(喜神) | 용신을 돕는 오행 |
| 기신(忌神) | 용신을 극하거나 방해하는 오행 |
| 구신(仇神) | 기신을 돕는 오행 |

## 6단계: 세부 운세 도출

### 6.1 운세 카테고리별 육친 연결

| 운세 항목 | 관련 육친 | 관련 오행(예: 무토일간) |
|:---|:---|:---|
| 직장운 | 관성(官星) | 木 |
| 금전운 | 재성(財星) | 水 |
| 연애운(남) | 재성(財星) | 水 |
| 연애운(여) | 관성(官星) | - |
| 건강운 | 일간 + 용신 상태 | 土 + 金 |
| 학업운 | 인성(印星) | 火 |

### 6.2 세부 운세 계산 로직

```
세부운세 점수 = 기본점수(3점)
              + 세운 오행이 해당 육친을 생하면 (+1)
              + 세운 오행이 해당 육친이면 (+1)
              + 세운 오행이 해당 육친을 극하면 (-1)
              + 용신과 일치하면 (+1)
              + 기신과 일치하면 (-1)
```

### 6.3 계산 예시: 무토일간의 2026년 금전운

```
금전운 관련 육친: 재성(水)
세운: 병오(丙午) = 火

火는 水를 극함 → 재성이 손상 → -1점
火는 기신(인성 과다) → -1점
기본 점수 3 - 1 - 1 = 1점 (★★☆☆☆)
```

## 9단계: 해석 텍스트 생성 로직

### 9.1 조건별 문장 템플릿

**총운 템플릿 예시:**

```
IF 세운 = 용신 THEN
  "[연도]년은 용신 [오행]이 들어오는 해로, 전반적으로 운세가 상승합니다."

IF 세운 = 기신 THEN
  "[연도]년은 기신 [오행]이 강한 해로, 신중한 처신이 필요합니다."

IF 세운 육친 = 관성 THEN
  "직장과 사회적 위상에 변화가 예상됩니다. 승진이나 새로운 책임이 주어질 수 있습니다."

IF 세운 육친 = 인성 THEN
  "학습과 자기계발에 좋은 해입니다. 자격증 취득이나 새로운 공부를 시작하기 좋습니다."

IF 세운 육친 = 재성 THEN
  "재물과 인연에 변화가 있는 해입니다. 새로운 수입원이나 인연이 생길 수 있습니다."

IF 세운 육친 = 식상 THEN
  "표현력과 창의력이 빛나는 해입니다. 자신의 재능을 펼칠 기회가 옵니다."

IF 세운 육친 = 비겁 THEN
  "경쟁과 협력이 공존하는 해입니다. 동료나 형제와의 관계에 신경 쓰세요."
```

### 9.2 세부 운세 템플릿 예시

**금전운:**

```
IF 재성 생조됨 THEN
  "재물운이 상승합니다. 새로운 수입원이 생기거나 투자에 좋은 시기입니다."

IF 재성 극함 THEN
```
