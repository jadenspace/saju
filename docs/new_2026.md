# 📄 PRD: 2026 병오년 신년운세 서비스

---

## 📋 문서 정보

| 항목 | 내용 |
|:---|:---|
| **프로젝트명** | 2026 병오년 신년운세 (丙午年 新年運勢) |
| **버전** | v1.0 |
| **작성일** | 2024년 12월 |
| **목적** | 무료 신년운세로 고객 유입 → 유료 용신 분석 전환 |

## 1. 서비스 개요

### 1.1 비즈니스 목표

```
┌─────────────────────────────────────────────────────────┐
│                    퍼널 전략                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   [STEP 1] 신년운세 무료 제공                          │
│            ↓                                           │
│   [STEP 2] 신뢰도 확보 (상세하고 정확한 분석)          │
│            ↓                                           │
│   [STEP 3] "왜 이런 운세인지 알고 싶다면?"             │
│            ↓                                           │
│   [STEP 4] 유료 용신 분석 전환                         │
│                                                         │
│   목표 전환율: 무료 → 유료 3~5%                       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 1.2 입력 정보

| 입력 항목 | 형식 | 용도 |
|:---|:---|:---|
| **생년** | 1940~2010 | 년주 산출, 대운 계산 |
| **생월** | 1~12월 (양/음력) | 월주 산출, 월령 판단 |
| **생일** | 1~31일 | 일주 산출 (일간 = 나) |
| **생시** | 시간 또는 모름 | 시주 산출 (선택) |
| **성별** | 남/여 | 대운 순행/역행 결정 |

### 1.3 출력 구성

```
┌─────────────────────────────────────────────────────────┐
│                    운세 출력 구조                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   1. 📊 2026년 총운 (Total Fortune)                    │
│      └─ 전체적인 한 해의 흐름과 테마                   │
│                                                         │
│   2. 💰 재물운 (Wealth)                                │
│      └─ 수입, 투자, 소비, 재테크 방향                 │
│                                                         │
│   3. 💕 애정운 (Love)                                  │
│      └─ 연애, 결혼, 부부관계, 이성운                  │
│                                                         │
│   4. 💼 직장운 (Career)                                │
│      └─ 취업, 승진, 이직, 사업운                      │
│                                                         │
│   5. 🏥 건강운 (Health)                                │
│      └─ 주의할 질병, 건강 관리 방향                   │
│                                                         │
│   6. 📅 월별운 (Monthly - 12개월)                      │
│      └─ 1월~12월 각 월의 운세 흐름                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 핵심 분석 키: 일간(日干)과 세운(歲運)의 관계

```
┌─────────────────────────────────────────────────────────┐
│              운세 분석의 핵심 공식                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   운세 = f(일간, 원국, 대운, 세운, 월운)               │
│                                                         │
│   • 일간(日干): 나 자신 = 분석의 기준점                │
│   • 원국(原局): 타고난 사주 8자 = 기본 성향            │
│   • 대운(大運): 10년 단위 큰 흐름 = 시대적 배경        │
│   • 세운(歲運): 해당 연도 = 2026년 丙午               │
│   • 월운(月運): 해당 월 = 1~12월 각각의 간지          │
│                                                         │
│   ──────────────────────────────────────────────────   │
│                                                         │
│   핵심 질문:                                           │
│   "2026년 丙午가 나의 일간에게 어떤 십신(十神)인가?"   │
│                                                         │
│   예시: 일간 戊土 기준                                 │
│   • 丙火(세운 천간) → 戊土의 편인(偏印)               │
│   • 午火(세운 지지) → 戊土의 편인(偏印) 통근          │
│   → 2026년은 "편인의 해" = 학습, 자격증, 변화의 해    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 3. 운세 항목별 분석 로직

### 3.1 총운 (總運) 분석

#### 3.1.1 분석 요소

| 분석 요소 | 사주 활용 | 산출 방법 |
|:---|:---|:---|
| **세운 십신** | 일간 vs 세운 간지 | 2026 丙午가 일간에게 어떤 십신인지 |
| **대운 십신** | 일간 vs 현재 대운 | 현재 대운이 일간에게 어떤 십신인지 |
| **세운-원국 작용** | 세운 vs 원국 8자 | 충/합/형/파/해 관계 |
| **세운-대운 작용** | 세운 vs 대운 | 충/합 관계 |
| **신강신약 고려** | 원국 분석 | 신강이면 설기/극 좋음, 신약이면 생/조 좋음 |

#### 3.1.2 총운 판단 알고리즘

```
┌─────────────────────────────────────────────────────────┐
│                 총운 판단 로직                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  STEP 1: 세운 십신 확인                                │
│  ─────────────────────────────                         │
│  일간이 戊土일 때, 2026 丙午는?                        │
│  • 丙 → 戊의 편인(偏印)                               │
│  • 午 → 丁火 정인 + 己土 겁재 (지장간)                │
│  → 주요 키워드: 인성(학습, 문서, 자격증, 어머니)       │
│                                                         │
│  STEP 2: 신강/신약에 따른 해석                         │
│  ─────────────────────────────                         │
│  IF 신약 사주:                                         │
│     인성(丙午)은 희신 → 좋은 해                       │
│     "도움을 받는 해, 학습에 유리, 후원자 등장"         │
│                                                         │
│  IF 신강 사주:                                         │
│     인성(丙午)은 기신 → 주의 필요                     │
│     "생각이 많아지는 해, 게으름 주의, 실행력 필요"     │
│                                                         │
│  STEP 3: 원국과의 작용 확인                            │
│  ─────────────────────────────                         │
│  • 午-子 충(沖): 원국에 子가 있으면 충돌 → 변동수     │
│  • 午-未 합(合): 원국에 未가 있으면 합 → 안정        │
│  • 午-午 자형(自刑): 원국에 午가 있으면 주의          │
│  • 寅-午-戌 삼합: 원국에 寅,戌 있으면 화국 → 강화    │
│                                                         │
│  STEP 4: 대운과의 조합                                 │
│  ─────────────────────────────                         │
│  현재 대운 + 세운 조합으로 시너지/충돌 판단            │
│  예: 대운 壬水 + 세운 丙火 = 충(沖) → 변화 많은 해    │
│                                                         │
│  STEP 5: 총운 등급 산출                                │
│  ─────────────────────────────                         │
│  점수화: 희신 작용 (+), 기신 작용 (-), 충/합 가감     │
│  → 상상(上上) / 상(上) / 중상(中上) / 중(中) /        │
│     중하(中下) / 하(下) / 하하(下下)                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 3.2 재물운 (財物運) 분석

#### 3.2.1 분석 요소

| 분석 요소 | 사주 활용 | 의미 |
|:---|:---|:---|
| **원국 재성 상태** | 정재/편재 유무, 위치, 강약 | 타고난 재물 복 |
| **세운과 재성 관계** | 2026 丙午 vs 원국 재성 | 올해 재물 흐름 |
| **세운이 재성인가?** | 일간 기준 세운 십신 | 세운 자체가 재성이면 재물 기회 |
| **식상 → 재성 흐름** | 식상생재 여부 | 노력으로 돈 버는 구조 |
| **비겁과 재성 충돌** | 비겁쟁재 여부 | 경쟁, 손재 가능성 |

#### 3.2.2 재물운 판단 알고리즘

```
┌─────────────────────────────────────────────────────────┐
│                 재물운 판단 로직                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  STEP 1: 일간 기준 재성(財星) 확인                     │
│  ─────────────────────────────                         │
│  일간별 재성:                                          │
│  ┌────────┬────────────┬────────────┐                 │
│  │ 일간   │ 정재(正財) │ 편재(偏財) │                 │
│  ├────────┼────────────┼────────────┤                 │
│  │ 甲乙木 │ 己土       │ 戊土       │                 │
│  │ 丙丁火 │ 辛金       │ 庚金       │                 │
│  │ 戊己土 │ 癸水       │ 壬水       │                 │
│  │ 庚辛金 │ 乙木       │ 甲木       │                 │
│  │ 壬癸水 │ 丁火       │ 丙火       │                 │
│  └────────┴────────────┴────────────┘                 │
│                                                         │
│  STEP 2: 2026 丙午가 재성인지 확인                     │
│  ─────────────────────────────                         │
│  예: 일간이 壬水인 경우                                │
│  • 丙火 = 壬水의 편재(偏財) ✓                         │
│  • 午火(丁) = 壬水의 정재(正財) ✓                     │
│  → 2026년은 "재성의 해" = 재물 기회 多                │
│                                                         │
│  예: 일간이 戊土인 경우                                │
│  • 丙火 = 戊土의 편인(偏印) ✗ 재성 아님              │
│  → 2026년 세운 자체는 재성 아님                       │
│  → 원국 재성 상태와 세운 작용으로 판단                │
│                                                         │
│  STEP 3: 원국 재성과 세운의 작용                       │
│  ─────────────────────────────                         │
│  • 세운이 원국 재성을 생(生)하는가?                   │
│    예: 원국 壬水(재성) ← 세운 庚金이 생해줌 → 재물↑  │
│                                                         │
│  • 세운이 원국 재성을 극(剋)하는가?                   │
│    예: 원국 壬水(재성) ← 세운 戊土가 극함 → 재물↓   │
│                                                         │
│  • 세운이 원국 재성과 합(合)/충(沖)하는가?            │
│    예: 원국 子水(재성) ← 세운 午火와 충 → 변동       │
│                                                         │
│  STEP 4: 신강/신약에 따른 재성 해석                    │
│  ─────────────────────────────                         │
│  IF 신강 + 재성 강화:                                  │
│     "재물을 감당할 힘이 있음 → 큰 돈 기회"            │
│                                                         │
│  IF 신약 + 재성 강화:                                  │
│     "재물이 오히려 부담 → 과욕 금물, 분수 지킬 것"    │
│                                                         │
│  STEP 5: 식상생재 여부 확인                            │
│  ─────────────────────────────                         │
│  세운에서 식상 → 재성 흐름이 있는가?                  │
│  → 있다면: "노력/기술/재능으로 돈 버는 구조"          │
│                                                         │
│  STEP 6: 재물운 등급 및 월별 흐름                      │
│  ─────────────────────────────                         │
│  재물운: 상/중상/중/중하/하                           │
│  재물 상승 시기: {월운에서 재성 강화되는 달}          │
│  재물 주의 시기: {월운에서 재성 충/극 당하는 달}      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
