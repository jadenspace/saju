# 일주 분석 데이터 생성 프롬프트

## 목적
60갑자 일주별 남녀 성격 분석 데이터를 생성하여 사주 애플리케이션에 사용

## 요청 형식

```
다음 일주들에 대한 분석 데이터를 TypeScript 형식으로 작성해주세요.

일주 목록: [일주 한자 5-10개]

각 일주마다 다음 형식을 따라주세요:

{
  ilju: '甲子',
  male: {
    summary: '(한 줄 요약 - 30-50자)',
    personality: [
      '(주요 성격 특징 1)',
      '(주요 성격 특징 2)',
      '(주요 성격 특징 3)',
      '(주요 성격 특징 4)'
    ],
    strengths: [
      '(장점 1)',
      '(장점 2)',
      '(장점 3)'
    ],
    weaknesses: [
      '(단점/주의사항 1)',
      '(단점/주의사항 2)',
      '(단점/주의사항 3)'
    ],
    career: '(적합한 직업 - 구체적으로)',
    wealth: '(재물운 - 2-3문장)',
    relationships: '(대인관계/결혼 - 2-3문장)'
  },
  female: {
    // 동일한 구조
  }
}

작성 가이드라인:
1. summary는 해당 일주의 핵심 특성을 한 문장으로 압축
2. personality는 구체적이고 차별화된 특징 (긍정+부정 균형)
3. strengths는 실제 장점 위주
4. weaknesses는 조심해야 할 점 (너무 부정적이지 않게)
5. career는 구체적인 직업군 나열
6. wealth는 재물운의 특성과 조언
7. relationships는 연애/결혼 스타일과 배우자 관계

참고할 기존 데이터:
[기존 데이터 1-2개 예시 첨부]

천간 의미:
- 甲(갑): 큰 나무, 양목, 리더십, 진취적
- 乙(을): 작은 나무, 음목, 유연함, 섬세함
- 丙(병): 태양, 양화, 밝음, 열정
- 丁(정): 촛불, 음화, 따뜻함, 예의
- 戊(무): 산, 양토, 책임감, 안정
- 己(기): 밭, 음토, 포용력, 실리
- 庚(경): 쇠, 양금, 강함, 결단력
- 辛(신): 보석, 음금, 세련됨, 예민함
- 壬(임): 바다, 양수, 지혜, 활동적
- 癸(계): 이슬, 음수, 섬세함, 감성

지지 의미:
- 子(자): 겨울 물, 지혜
- 丑(축): 겨울 흙, 신중
- 寅(인): 봄 나무, 활력
- 卯(묘): 봄 나무, 온화
- 辰(진): 봄 흙, 변화
- 巳(사): 여름 불, 영리
- 午(오): 여름 불, 열정
- 未(미): 여름 흙, 부드러움
- 申(신): 가을 금, 강인함
- 酉(유): 가을 금, 섬세함
- 戌(술): 가을 흙, 충성
- 亥(해): 겨울 물, 순수

남녀 차이:
- 남자: 사회적 성취, 리더십, 커리어 중심
- 여자: 대인관계, 감성, 가정과 커리어 균형
- 배우자 관계 해석 시 성별 고려
```

## 배치 처리 전략

### 1단계: 10간 × 본궁 (10개)
천간과 같은 지지 조합
- 甲寅, 乙卯, 丙午, 丁巳, 戊戌, 己未, 庚申, 辛酉, 壬子, 癸亥

### 2단계: 주요 일주 (10개)
자주 분석되는 일주
- 甲子, 乙丑, 丙寅, 丁卯, 戊辰, 己巳, 庚午, 辛未, 壬申, 癸酉

### 3단계: 나머지 일주 (40개)
10개씩 배치로 요청

## 효율적인 프롬프트 예시

```
다음 10개 일주에 대한 남녀별 분석 데이터를 작성해주세요.
TypeScript 객체 배열 형식으로, 즉시 코드에 붙여넣을 수 있도록 해주세요.

일주: 甲戌, 甲申, 甲午, 甲辰, 甲寅, 乙亥, 乙酉, 乙未, 乙巳, 乙卯

형식:
[
  {
    ilju: '甲戌',
    male: {
      summary: '...',
      personality: ['...', '...', '...', '...'],
      strengths: ['...', '...', '...'],
      weaknesses: ['...', '...', '...'],
      career: '...',
      wealth: '...',
      relationships: '...'
    },
    female: {
      // 동일 구조
    }
  },
  // ... 나머지 9개
]

참고:
- 천간과 지지의 상생상극 관계 고려
- 각 일주의 독특한 특성 부각
- 실제 사주 이론 기반
- 긍정적이되 현실적인 톤
```

## 검증 체크리스트

생성된 데이터를 검토할 때:
- [ ] 각 일주마다 male, female 모두 있는가?
- [ ] summary가 해당 일주의 특성을 잘 반영하는가?
- [ ] personality 4개가 모두 차별화되는가?
- [ ] 같은 문구가 반복되지 않는가?
- [ ] 직업이 구체적인가?
- [ ] 재물운과 관계운이 2-3문장으로 충실한가?
- [ ] TypeScript 문법이 올바른가?
- [ ] 천간과 지지의 의미가 조화로운가?
