# 신년운세 리포트 출력 설계안

## 1. 전체 구조 및 데이터 흐름

```
┌──────────────────────┐
│  생년월일시 입력        │
│  예: 1990-03-15-10시  │
└──────────────────────┘
         ↓
┌──────────────────────┐
│  사주팔자 자동 계산     │
│  - 연주: 庚午          │
│  - 월주: 己卯          │
│  - 일주: 甲寅 ← 핵심   │
│  - 시주: 己巳          │
└──────────────────────┘
         ↓
┌──────────────────────┐
│  2026년 병오년 매칭    │
│  - 일간(甲) vs 천간(丙) → 식신  │
│  - 일지(寅) vs 지지(午) → 인오반합 │
└──────────────────────┘
         ↓
┌──────────────────────┐
│  신년운세 리포트 생성   │
│  (아래 2번 구조)       │
└──────────────────────┘
```

## 2. 신년운세 리포트 세부 설계

### 2-1. 총운 점수

#### 📊 점수 산출 기준

**점수 체계: 100점 만점**

| 요소 | 배점 | 산출 방법 |
|------|------|-----------|
| 십성 길흉도 | 40점 | 재성/인성(40점) > 식상(35점) > 비겁(25점) > 관성(20점) |
| 형충화합 | 30점 | 합(30점) > 반합(25점) > 비화(20점) > 무(15점) > 해(10점) > 자형(5점) > 충(0점) |
| 사주 조화도 | 20점 | 오행 균형도 계산 (火 과다면 감점) |
| 계절 적합도 | 10점 | 출생월과 세운의 조화 |

#### 점수 구간 및 등급

| 점수 | 등급 | 한줄평 |
|------|------|--------|
| 90-100 | S (최상) | 🌟🌟🌟🌟🌟 대길! 적극적 행동의 해 |
| 80-89 | A (상) | 🌟🌟🌟🌟 길! 기회가 많은 해 |
| 70-79 | B+ (중상) | 🌟🌟🌟 평길! 안정적 발전의 해 |
| 60-69 | B (중) | 🌟🌟 평! 꾸준함이 필요한 해 |
| 50-59 | C (중하) | 🌟 소심! 신중함이 필요한 해 |
| 40-49 | D (하) | ⚠️ 주의! 방어적 자세 필요 |
| 0-39 | E (최하) | ⚠️⚠️ 경고! 큰 결정 자제 |

#### 📝 출력 예시

```
┌─────────────────────────────────────┐
│         2026년 을사님의 총운          │
├─────────────────────────────────────┤
│                                     │
│            ⭐ 85점 / A등급            │
│                                     │
│    🌟🌟🌟🌟 길! 기회가 많은 해       │
│                                     │
├─────────────────────────────────────┤
│  【점수 산출 내역】                   │
│  ✓ 십성 길흉도: 35/40점 (식신)       │
│  ✓ 형충화합: 25/30점 (인오반합)      │
│  ✓ 사주 조화도: 18/20점              │
│  ✓ 계절 적합도: 7/10점               │
├─────────────────────────────────────┤
│  【해석】                            │
│  올해는 여러분의 재능과 창의성이      │
│  빛을 발하는 한 해입니다. 특히 새로운 │
│  프로젝트나 사업 확장에 유리하며,     │
│  노력한 만큼 성과를 볼 수 있습니다.   │
│  상반기(1~6월)가 특히 좋으니 이 시기에│
│  중요한 일을 추진하시기 바랍니다.     │
└─────────────────────────────────────┘
```

### 2-2. 세부 운세 (재물/직업/이성/건강)

#### 🎯 영역별 점수 산출 기준

**각 영역별 100점 만점**

| 영역 | 핵심 판단 기준 | 가중치 |
|------|----------------|--------|
| 재물운 | 재성(財星) 유무 및 강도 | 십성 60% + 형충화합 40% |
| 직업운 | 관성(官星) + 식상(食傷) | 십성 70% + 형충화합 30% |
| 이성운 | 재성(남성) / 관성(여성) + 합(合) | 십성 50% + 형충화합 50% |
| 건강운 | 충·형·해 + 오행 균형 | 충돌 여부 60% + 오행 40% |

#### 📊 십성별 영역 점수표

##### 재물운 점수

| 십성 | 기본 점수 | 형충화합 보너스 | 최종 범위 |
|------|-----------|-----------------|-----------|
| 재성(財星) | 90 | 합(+10), 반합(+5) | 95-100 |
| 식상(食傷) | 75 | 생재 구조면 +10 | 75-85 |
| 인성(印星) | 60 | - | 60-65 |
| 비겁(比劫) | 40 | 충(-10) | 30-40 |
| 관성(官星) | 50 | 관살혼잡 -10 | 40-50 |

##### 직업운 점수

| 십성 | 기본 점수 | 형충화합 보너스 | 최종 범위 |
|------|-----------|-----------------|-----------|
| 관성(官星) | 85 | 합(+10) | 85-95 |
| 식상(食傷) | 80 | 반합(+10) | 80-90 |
| 인성(印星) | 75 | - | 75-80 |
| 재성(財星) | 70 | - | 70-75 |
| 비겁(比劫) | 50 | 충(-15) | 35-50 |

##### 이성운 점수

| 십성 | 남성 | 여성 | 형충화합 보너스 |
|------|------|------|-----------------|
| 재성(財星) | 90 | 60 | 합(+10) |
| 관성(官星) | 60 | 90 | 합(+10) |
| 식상(食傷) | 70 | 55 | - |
| 인성(印星) | 50 | 50 | - |
| 비겁(比劫) | 40 | 40 | 충(-20) |

##### 건강운 점수

| 형충화합 | 기본 점수 | 오행 균형도 반영 |
|----------|-----------|------------------|
| 무관계 | 85 | +5 (균형) / -5 (불균형) |
| 합 | 80 | - |
| 반합 | 75 | - |
| 비화 | 70 | -5 (과다) |
| 해 | 60 | - |
| 자형 | 50 | -10 |
| 충 | 40 | -15 |

### 2-4. 기회 월 출력

#### 🎯 기회 월 선정 기준

**선정 조건 (3개 이상 해당 시)**

1. 월별 점수 90점 이상
2. 형충화합이 "합" 또는 "삼합"
3. 십성이 재성 또는 관성
4. 세부 운세 중 2개 이상 80점 이상
