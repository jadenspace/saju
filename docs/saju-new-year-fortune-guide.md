# 사주명리학 신년운세 분석 가이드 (서비스 확장판)

> 본 문서는 사주명리학 기반 **신년운세 서비스 구현**을 위한 도메인 + 해석 구조 가이드입니다.  
> 단순 이론 설명을 넘어, **사용자가 ‘신통하다’고 느끼는 운세 출력 구조**를 목표로 합니다.

---

## 1. 신년운세 해석의 서비스 원칙 (신규)

### 1.1 기존 신년운세의 한계
- 점수 중심 → 근거 부족
- 십성·오행 중심 설명 → 비전문가 이해 어려움
- 매년 비슷한 문장 → 신뢰도 저하

### 1.2 개선된 해석 원칙

신년운세는 다음 3단 구조로 해석되어야 한다.

**[명리 계산] → [운의 작동 방식] → [체감 문장]**

- **명리 계산**: 내부 로직 (십성, 합충형파, 용신 등)
- **작동 방식**: 데이터 레벨에서 운이 어떻게 발현되는지 정의
- **체감 문장**: 사용자에게 노출되는 부드러운 텍스트

---

## 2. 신년운세 전용 핵심 개념 (신규)

### 2.1 “운의 작동 방식” 개념 도입

> 운은 좋고 나쁨이 아니라 **어떻게 작동하느냐**의 문제다.

#### 예시
- 편인 多 → ❌ “편인운이 들어왔습니다.”
- 편인 多 → ✅ “감정보다 판단과 분석이 먼저 작동하는 해입니다.”

```typescript
yearMechanism: {
  type: "thinking-first",
  description: "감정보다 판단과 분석이 먼저 작동하는 해",
  advantage: ["분석력", "정리력", "기획"],
  risk: ["결정 지연", "고립"]
}
```

### 2.2 “흔히 겪는 실수 패턴” (신규)
신통한 운세는 사용자의 가려운 곳이나 불편한 지점을 정확히 찌른다.

```typescript
commonMistake: {
  title: "생각만 하다 타이밍을 놓치기 쉬운 해",
  situations: [
    "중요한 결정을 계속 뒤로 미룸",
    "완벽한 조건이 갖춰질 때까지 기다림",
    "주변 피드백 없이 배울 점 없는 고민만 반복"
  ]
}
```

## 3. 총운(總運) 해석 구조 수정

### 기존
- 총운 점수 + 짧은 설명 문장

### 개선
```typescript
yearSummary: {
  score: 75,
  summaryText: "생각이 곧 기회가 되는 해",
  reason: [
    "사고 중심 십성(인성) 활성화",
    "원국과 세운 사이의 큰 충 없이 안정적인 에너지 흐름"
  ]
}
```
**사용자 노출 예시:**
> 2026년은 삶이 크게 흔들리지는 않지만, 스스로의 판단과 선택 하나하나가 실제 결과로 이어지는 해입니다.

## 5. 월별 운세 해석 방식 수정

### 기존
- 12개월 전체에 대해 동일한 비중의 분석 노출

### 개선 원칙
- **특징적인 달만 추출**: 기복이 크거나 기회가 있는 달을 강조
- **“왜 중요한지”를 테마로 요약**

```typescript
keyMonths: [
  {
    month: 3,
    theme: "판단 과부하 주의",
    advice: "결정 기준을 단순화하고 기록에 의존하세요."
  },
  {
    month: 7,
    theme: "관계 소홀로 인한 고립",
    advice: "가까운 사람에게 먼저 마음을 표현하는 것이 중요합니다."
  }
]
```

## 6. “운을 잘 쓰는 법” 데이터화 (신규)

```typescript
fortuneGuide: {
  do: ["생각을 바로 기록하기", "중간 과정을 동료와 공유", "충분히 정리 후 최종 결정"],
  dont: ["마지막까지 결정 미루기", "고민을 혼자서만 끌어안고 있기"],
  keywords: ["체계적 정리", "냉철한 선택", "사고력의 확장"]
}
```
- UI의 체크리스트, 푸시 알림, 또는 AI 문장 생성의 핵심 소스로 재사용합니다.

## 8. 기존 명리 이론 파트 (계산 로직 참조용)

본 섹션은 실제 계산 엔진 구현을 위한 기술적 참조 데이터입니다. 사용자에게 노출 시 다음 가이드를 반드시 준수하십시오.

### ⚠️ 사용자 노출 시 주의사항
- **십성명 직접 노출 ❌**: '편인운' 대신 '생각이 깊어지는 시기'로 표현
- **“~의 기운” → “~한 흐름”으로 변환**: 추상적 표현 지양
- **점수 단독 노출 ❌**: 항상 그 점수가 도출된 이유를 함께 설명

## 9. 핵심 요약
신년운세의 완성도는 **명리 정확도 50% + 해석 구조 50%**가 결정합니다.
이미 계산 로직은 충분히 정교하므로, 이제는 사용자가 '신통하다'고 느끼를 수 있는 **데이터 가공과 구조화**에 집중해야 합니다.
