# 사주명리학 신년운세 분석 가이드

> 본 문서는 사주명리학 기반 신년운세 분석 시스템 개발을 위한 도메인 지식 문서입니다.
> 참고 출처: 도화도르(도화로운) 유튜브 채널 및 사주명리학 자료

---

## 1. 개요

### 1.1 신년운세란?
신년운세는 사주팔자(四柱八字)를 기반으로 특정 해에 개인에게 작용하는 운의 흐름을 분석하는 것입니다. 단순한 띠별 운세가 아니라, 개인의 사주 원국(原局)과 그 해의 세운(歲運)이 어떻게 상호작용하는지를 종합적으로 분석합니다.

### 1.2 핵심 원리
- **사주 원국**: 태어난 년, 월, 일, 시의 천간(天干)과 지지(地支) 8글자
- **세운(歲運)**: 해당 연도의 천간과 지지 (예: 2025년 = 을사년 乙巳年)
- **상호작용**: 원국의 글자들과 세운의 글자가 만나 생기는 합(合), 충(沖), 형(刑), 파(破) 등의 작용

## 2. 운(運)의 체계

### 2.1 운의 4가지 종류

| 운 종류 | 작용 기간 | 설명 | 중요도 |
|---------|----------|------|--------|
| **대운(大運)** | 10년 | 인생의 큰 흐름을 결정하는 운 | ⭐⭐⭐⭐⭐ |
| **세운(歲運)/년운** | 1년 | 해당 연도에 작용하는 운 (신년운세의 핵심) | ⭐⭐⭐⭐ |
| **월운(月運)** | 1개월 | 해당 월에 작용하는 운 | ⭐⭐⭐ |
| **일운(日運)** | 1일 | 해당 일에 작용하는 운 | ⭐⭐ |

### 2.2 대운 상세
```
- 대운은 10년 주기로 바뀜
- 대운 전반 5년: 천간(위 글자)이 강하게 작용
- 대운 후반 5년: 지지(아래 글자)가 강하게 작용
- 교운기(대운이 바뀌는 시기): 약 6개월~1년 전부터 변화 시작
```

## 3. 오행(五行) 체계

### 3.1 오행별 속성

| 오행 | 한자 | 자연물 | 색상 | 계절 | 방위 |
|------|------|--------|------|------|------|
| 목(木) | 木 | 나무 | 초록색/청색 | 봄 | 동 |
| 화(火) | 火 | 불 | 빨간색 | 여름 | 남 |
| 토(土) | 土 | 흙/땅 | 노란색 | 환절기 | 중앙 |
| 금(金) | 金 | 쇠/금속 | 흰색 | 가을 | 서 |
| 수(水) | 水 | 물 | 검은색 | 겨울 | 북 |

### 3.2 오행의 상생상극

```
상생(相生) - 서로 돕는 관계:
목 → 화 → 토 → 금 → 수 → 목
(나무가 불을 생하고, 불이 흙을 생하고...)

상극(相剋) - 서로 제어하는 관계:
목 → 토 → 수 → 화 → 금 → 목
(나무가 흙을 극하고, 흙이 물을 극하고...)
```

## 6. 신년운세 분석 항목

### 6.1 필수 분석 항목

| 항목 | 설명 | 분석 방법 |
|------|------|----------|
| **총운** | 1년 전체 흐름 | 세운과 원국의 전체적 조화 분석 |
| **재물운** | 금전, 재산 | 재성(편재, 정재)의 작용 분석 |
| **연애운/애정운** | 이성, 배우자 | 남: 재성 / 여: 관성 분석 |
| **직업운** | 취업, 승진, 사업 | 관성, 식상 분석 |
| **건강운** | 신체, 정신 건강 | 오행 균형, 충극 분석 |
| **학업운** | 시험, 자격증 | 인성(정인, 편인) 분석 |

### 6.2 월별 운세 분석

각 월의 천간지지와 세운, 원국의 상호작용을 분석합니다.

```
월별 분석 = f(원국, 대운, 세운, 월운)

예시: 2025년 3월
- 세운: 을사(乙巳) - 목+화
- 월운: 기묘(己卯) - 토+목
- 분석: 원국과의 합충형파 + 십성 작용
```

## 7. 간지 작용 (합충형파)

### 7.1 천간 작용

**천간합(天干合)**
```
갑(甲) + 기(己) = 토(土)
을(乙) + 경(庚) = 금(金)
병(丙) + 신(辛) = 수(水)
정(丁) + 임(壬) = 목(木)
무(戊) + 계(癸) = 화(火)
```

**천간충(天干沖)**
```
갑(甲) ↔ 경(庚)
을(乙) ↔ 신(辛)
병(丙) ↔ 임(壬)
정(丁) ↔ 계(癸)
무(戊) ↔ 갑(甲) (일부 학파)
```

### 7.2 지지 작용

**삼합(三合)**
```
인오술(寅午戌) → 화(火)
사유축(巳酉丑) → 금(金)
신자진(申子辰) → 수(水)
해묘미(亥卯未) → 목(木)
```

**방합(方合)**
```
인묘진(寅卯辰) → 동방 목(木)
사오미(巳午未) → 남방 화(火)
신유술(申酉戌) → 서방 금(金)
해자축(亥子丑) → 북방 수(水)
```

**육합(六合)**
```
자축합(子丑合) → 토
인해합(寅亥合) → 목
묘술합(卯戌合) → 화
진유합(辰酉合) → 금
사신합(巳申合) → 수
오미합(午未合) → 화/토
```

**육충(六沖)**
```
자오충(子午沖)
축미충(丑未沖)
인신충(寅申沖)
묘유충(卯酉沖)
진술충(辰戌沖)
```

## 10. 분석 알고리즘 설계 가이드

### 10.1 신년운세 분석 플로우

```
1. 입력 데이터 수집
   - 생년월일시 (양력/음력)
   - 성별
   - 분석 대상 연도

2. 사주 원국 생성
   - 만세력 기반 사주팔자 계산
   - 대운 계산 (성별에 따라 순행/역행)

3. 세운 정보 생성
   - 대상 연도의 천간지지
   - 월별 천간지지

4. 상호작용 분석
   - 합충형파 분석
   - 십성 작용 분석
   - 신살 작용 분석

5. 영역별 운세 생성
   - 총운, 재물운, 연애운, 직업운, 건강운, 월별운
   
6. 결과 출력
   - 종합 해석
   - 조언 및 주의사항
```

### 10.2 핵심 계산 함수 (의사코드)

```javascript
// 일간 기준 십성 계산
function calculateTenGods(dayMaster, targetElement) {
  const relation = getRelation(dayMaster, targetElement);
  const sameYinYang = isSameYinYang(dayMaster, targetElement);
  
  switch(relation) {
    case 'same': return sameYinYang ? '비견' : '겁재';
    case 'iGenerate': return sameYinYang ? '식신' : '상관';
    case 'iControl': return sameYinYang ? '편재' : '정재';
    case 'controlsMe': return sameYinYang ? '편관' : '정관';
    case 'generatesMe': return sameYinYang ? '편인' : '정인';
  }
}

// 운세 점수 계산 (예시)
function calculateFortuneScore(originalChart, yearPillar) {
  let score = 50; // 기본 점수
  
  // 합이 있으면 가점
  if (hasHarmony(originalChart, yearPillar)) score += 15;
  
  // 충이 있으면 감점
  if (hasClash(originalChart, yearPillar)) score -= 20;
  
  // 용신이 들어오면 가점
  if (isYongShin(originalChart, yearPillar)) score += 25;
  
  // 기신이 들어오면 감점
  if (isGiShin(originalChart, yearPillar)) score -= 25;
  
  return Math.max(0, Math.min(100, score));
}
```
