# 용신 7단계 프로세스

---

## Step 1: 월령 심사 (月令 審査)

> 핵심 질문: "월령이 일간에게 어떤 존재인가?"
> 

---

### 1-1. 적천수 원문

**원문 1:**

> "月令乃提綱之府, 譬之宅也, 人元為用事之神"
(월령은 제강의 부이니 집에 비유하고, 인원은 용사의 신이다)
> 

**원문 2:**

> "五陽從氣不從勢, 五陰從勢無情義"
(오양은 기를 따르고 세를 따르지 않으며, 오음은 세를 따르되 정의가 없다)
> 

---

### 1-2. 원문 해석

### "月令乃提綱之府"

```
提綱 = 벼리(그물의 큰 줄), 강령
府 = 관청, 근본이 되는 곳

해석:
  월령은 사주 전체를 통솔하는 강령의 자리
  모든 판단의 출발점

```

### "譬之宅也"

```
譬 = 비유하다
宅 = 집

해석:
  월령을 집에 비유함
  → 일간이 어떤 환경(집)에 거주하는가
  → 그 집이 나에게 편한가, 불편한가

```

### "人元為用事之神"

```
人元 = 지장간 (지지 속에 숨은 천간)
用事 = 일을 주관함
神 = 주체

해석:
  지장간 중 사령하는 글자가 실제 주관자
  → 월지의 겉모습이 아니라 지장간을 봐야 함

```

#### 1-3-2. 원문 적용 분석

##### 조건 1: "月令乃提綱之府" 적용

```
질문: 월령 寅이 일간 戊에게 어떤 집인가?

寅 = 木의 자리
戊 = 土

木과 土의 관계: 木克土 (목이 토를 극함)

해석:
  戊土가 寅木의 집에 거주함
  → 나를 극하는 환경에 있음
  → 편하지 않은 집

```

##### 조건 2: 득령(得令) 여부 판단

```
【득령(得令)의 개념】

득령(得令): 일간이 월령에서 왕성한 상태
→ 월령이 일간의 왕지(旺地)이거나 생지(生地)일 때 득령

【이 사주 득령 여부】

일간: 戊土
월령: 寅 (木)

戊土가 寅월에서:
  - 寅 = 木의 왕지(旺地)
  - 戊土 입장에서 寅월은:
    * 절지(絶地) 또는 장생(長生)
    * (장생인지 절지인지는 학파에 따라 다름)
  - 어느 쪽이든 "득령하지 못함"은 명확

판단: ❌ 득령하지 못함
  → 戊土는 寅월에서 휴수(休囚)
  → 월령에서 실령(失令) 상태
  → 득령 여부: 득령 못함

```

##### 조건 3: 십신(十神) 배치

```
【십신(十神)의 개념】

십신: 일간을 기준으로 다른 천간/지지와의 관계
→ 월령의 십신 관계를 명확히 정리해야 함

【이 사주 월령의 십신 관계】

일간: 戊土 기준

월지 寅의 십신:
  - 寅(월지) = 편관(偏官) 궁(宮)
  - 寅의 지장간:
    * 戊(여기) = 비견(比肩) ✅
    * 丙(중기) = 편인(偏印) ✅
    * 甲(본기) = 편관(偏官) ⚠️

십신 관계 정리:
  - 월지 寅 = 편관 궁
  - 월지 본기 甲 = 편관 (나를 극함)
  - 월지 중기 丙 = 편인 (나를 생함)
  - 월지 여기 戊 = 비견 (나와 같은 편)

【십신 관계의 의미】

편관(甲木):
  - 나를 극하는 존재
  - 월령의 본기로 가장 강함
  - 불리함 ⚠️

편인(丙火):
  - 나를 생하는 존재
  - 월령의 중기로 투출함
  - 유리함 ✅

비견(戊土):
  - 나와 같은 편
  - 월령의 여기로 약함
  - 보조적 도움 ✅

```

##### 조건 4: "人元為用事之神" 적용

```
질문: 寅의 지장간 중 누가 용사(用事)하는가?

寅의 지장간: 戊 → 丙 → 甲

【중요】 사령(司令) 계산:
  적천수에서는 월령의 지장간이 순차적으로 사령함
  실제 생일(일간이 태어난 날짜)에 따라 용사지신이 달라짐

寅월의 사령 기간 (일반적 기준):
  - 戊土(여기): 입춘 후 7일간 사령
  - 丙火(중기): 다음 7일간 사령
  - 甲木(본기): 나머지 16일간 사령
  → 합계 30일 (총 일수가 맞음) ✅

【참고】 학파에 따른 차이:
  적천수 계열에서는 보통 위와 같이 봅니다.
  다만 일부 학파에서는:
  - 戊土: 7일
  - 丙火: 7일
  - 甲木: 약 16일 (또는 14~16일)
  → 학파에 따라 다소 차이가 있을 수 있음

이 사주 기준:
  - 생일이 언제인지에 따라 용사지신이 달라질 수 있음
  - 만약 생일이 戊土 사령 기간(입춘 후 7일)이면:
    → 용사지신은 戊土 (일간과 같은 편)
  - 만약 생일이 丙火 사령 기간(다음 7일)이면:
    → 용사지신은 丙火 (일간을 생함)
  - 만약 생일이 甲木 사령 기간(나머지 16일)이면:
    → 용사지신은 甲木 (일간을 극함)

【일반적 판단】:
  대부분의 경우 본기(本氣) 甲木이 사령 기간이 가장 길어
  전체적인 기운을 주도함
  → 甲木이 寅월을 주관하는 신(神)으로 봄

甲木 = 戊土의 편관 (나를 극하는 존재)

해석:
  월령을 주관하는 자가 나를 극하는 편관
  → 월령의 주인이 나에게 불리함
  → 단, 실제 생일 기준으로는 용사지신이 달라질 수 있음

```

##### 조건 5: "五陽從氣不從勢" / "五陰從勢無情義" 적용

```
【오양(五陽)과 오음(五陰)의 구분】

오양(五陽): 甲丙戊庚壬 (양간)
오음(五陰): 乙丁己辛癸 (음간)

적천수 원문:
  "五陽從氣不從勢" (오양은 기를 따르고 세를 따르지 않는다)
  "五陰從勢無情義" (오음은 세를 따르되 정의가 없다)

【이 사주 적용】

일간: 戊土 = 오양(五陽) 중 하나

"五陽從氣不從勢" 적용:
  戊土는 양간이므로:
  → 겉으로 보이는 세력(數)보다
  → 월령의 기운(氣)을 중시해야 함

  월령 寅의 기운: 木氣 (나를 극하는 기운)
  → 양간 戊土는 이 木氣의 영향을 강하게 받음

【참고】 만약 음간이었다면:

만약 일간이 己土(음간)였다면:
  "五陰從勢無情義" 적용
  → 겉으로 보이는 세력(數)을 중시
  → 월령의 기운(氣)보다 세력(數)을 따름
  → 다만 "정의가 없다"는 것은
     세를 따르되 그 세력이 일관되지 않을 수 있음을 의미

이 사주는 양간(戊土)이므로:
  → 기(氣)를 중시하는 판단이 맞음
  → 월령의 木氣를 강하게 받음

```

##### 조건 6: 월지와 다른 지지의 관계 (형충회합)

```
【형충회합의 개념】

월지(寅)와 다른 지지의 관계가
월령의 기운에 영향을 줄 수 있음
→ 특히 丙火(월간)의 근거지와의 관계가 중요

【이 사주의 지지 관계】

사주 지지: 巳(년지) 寅(월지) 戌(일지) 寅(시지)

① 寅과 巳의 관계:
  - 寅巳 상형(相刑) 또는 寅巳害
  - (학파에 따라 상형 또는 상해로 봄)
  - 巳는 丙火의 본기 통근지
  - 寅巳 형해로 인해:
    * 丙火의 뿌리(巳)가 약화될 수 있음 ⚠️
    * 다만 丙火가 월간에 투출하여 작용력 유지 ✅

② 寅과 戌의 관계:
  - 寅午戌 삼합(三合)의 일부
  - 午가 없어 완전한 삼합은 불성립
  - 寅戌 반합(半合) 가능성
  - 火局 형성 기운 (丙火와 관련)

③ 寅과 寅(시지)의 관계:
  - 자형(自刑) 또는 병렬
  - 같은 寅이 중복되어 木氣 강화
  - 甲木의 뿌리가 이중으로 강함

【형충회합이 월령에 미치는 영향】

긍정적 영향:
  - 寅戌 반합으로 火氣 모임
  - 丙火(월간)의 작용력 강화 가능 ✅

부정적 영향:
  - 寅巳 형해로 丙火의 뿌리 약화 가능 ⚠️
  - 다만 월간 투출로 상쇄됨 ✅

종합:
  - 형충회합이 월령의 기운에 영향
  - 특히 丙火의 작용력에 영향
  - Step 5에서 상세 분석 예정

```

#### 1-3-3. 월지 지장간 투출 분석

**적천수 관점:**

```
"人元為用事之神"
→ 지장간이 천간에 투출하면 그 작용이 드러남

```

##### 조건 7: 寅의 지장간 투출 여부 확인

**寅의 지장간 투출 여부:**

| 지장간 | 천간 투출 | 비고 |
| --- | --- | --- |
| 戊土 | ❌ | 己는 있으나 戊는 없음 |
| 丙火 | ✅ **월간** | 투출하여 작용 드러남 |
| 甲木 | ✅ **시간** | 투출하여 작용 드러남 |

**투출 강도 분석:**

```
【丙火 투출 강도 분석】

丙火의 위치: 월간 (월령에서 직접 투출)
→ 월령의 지장간이 월간으로 투출하는 것은 매우 강함

丙火의 뿌리(通根) 확인:
  - 巳(년지): 본기(本氣) 통근 ✅
    → 丙火가 巳에 본기로 뿌리를 둠
    → 강한 근거지
  - 寅(월지): 중기(中氣) 통근 ✅
    → 丙火가 寅의 지장간 중기로 통근
    → 추가적인 뿌리
  - 戌(일지): 통근 없음 ❌
    → 戌의 지장간은 辛(여기) → 丁(중기) → 戊(본기)
    → 丙火는 戌에 통근하지 않음

강도 판단:
  - 월간 투출 + 본기 통근(巳) + 중기 통근(寅)
  → 매우 강한 투출
  → 작용력이 크고 안정적임

【甲木 투출 강도 분석】

甲木의 위치: 시간 (時干)
→ 시간은 "무력지지(無力之地)"로 불림
→ 월령, 일지에 비해 힘이 약함

甲木의 뿌리(通根) 확인:
  - 寅(월지): 본기(本氣) 통근 ✅
    → 甲木이 寅에 본기로 뿌리를 둠
    → 강한 근거지
  - 寅(시지): 본기(本氣) 통근 ✅
    → 甲木이 시지에도 본기로 뿌리를 둠
    → 이중 뿌리로 매우 강함

강도 판단:
  - 시간 투출 (무력지지) ⚠️
  - 그러나 본기 통근이 이중(월지+시지)으로 매우 강함 ✅
  → 투출 위치는 약하나 뿌리가 강해 상쇄됨
  → 전체적으로 강한 작용력

【투출 강도 비교】

丙火 (월간):
  - 투출 위치: 월간 (강함) ✅
  - 통근: 본기(巳) + 중기(寅) ✅
  → 매우 강하고 안정적

甲木 (시간):
  - 투출 위치: 시간 (약함) ⚠️
  - 통근: 본기(寅×2) 이중 통근 ✅
  → 투출 위치는 약하나 뿌리가 강해 강함

```

```
판단:
  - 丙火(편인): 월간 투출 + 강한 통근 → 매우 강함
    → 나를 생하는 작용이 강하게 드러남
  - 甲木(편관): 시간 투출(무력지지)이나 이중 본기 통근
    → 나를 극하는 작용이 드러나나, 시간의 약함이 있음

  월령 속에 나를 돕는 것(丙)과 해치는 것(甲)이 공존
  둘 다 천간에 드러나 작용함
  → 다만 丙火가 위치상 더 유리함 (월간 vs 시간)

```

## Step 2: 조후 심사 (調候 審査)

> 핵심 질문: "천도의 한난(寒暖)이 일간에게 어떤 영향을 주는가?"
>

### 2-1. 적천수 원문

**원문:**

> "天道有寒暖, 發育萬物, 人道得之, 不可過也"
(천도에 한난이 있어 만물을 발육하니, 인도가 이를 얻되 지나침이 없어야 한다)
> 

---

### 2-2. 원문 해석

### "天道有寒暖"

```
天道 = 하늘의 이치, 자연의 법칙
寒暖 = 차가움과 따뜻함

해석:
  자연에는 춥고 따뜻한 기후가 있음
  → 계절에 따른 기후 변화
  → 월령(계절)이 가진 기후적 성질

```

### "發育萬物"

```
發育 = 발생하고 기름
萬物 = 만물

해석:
  한난의 조화가 만물을 자라게 함
  → 적절한 한난이 있어야 생명이 유지됨
  → 너무 춥거나 더우면 생육이 어려움

```

### "人道得之, 不可過也"

```
人道 = 사람의 도리, 사주에서는 일간
得之 = 이를 얻음
不可過 = 지나침이 없어야 함

해석:
  일간이 한난의 조화를 얻어야 함
  → 지나치게 춥거나 덥지 않아야 함
  → 극단적 기후는 일간에게 해로움

```

### 2-3. 이 사주에 적용

#### 2-3-1. 조후 분석

##### 조건 1: 월령의 한난(寒暖) 확인

```
월지: 寅 (정월, 입춘 후)

질문: 寅월의 천도는 어떠한가?

寅월의 기후:
  - 입춘 직후
  - 겨울이 끝나고 봄이 시작됨
  - 아직 한기(寒氣)가 남아있음

판단: 寒 (차가운 쪽)
  → "天道有寒暖" 중 寒에 해당

```

###### 조건 1-1: 寅월 戊土의 조후 우선순위

```
【《궁통보감》 원문】

> "正月戊土, 寒氣未除, 先丙解凍, 次甲疏土"
> (정월 무토는 한기가 아직 남아있으니, 
>  먼저 병화로 언 것을 녹이고, 
>  다음으로 갑목으로 흙을 풀어준다)

【원문 해석】

正月戊土:
  - 정월(寅월)의 戊土
  - 초춘(初春) 시기

寒氣未除:
  - 한기(寒氣)가 아직 제거되지 않음
  - 겨울의 차가움이 남아있음

先丙解凍:
  - "先"자: 먼저, 최우선
  - 丙火로 언 것을 녹임
  - 寒氣를 해소하는 것이 최우선

次甲疏土:
  - "次"자: 다음, 보조
  - 甲木으로 흙을 소토(疏土)
  - 丙火 다음 순위

【조후 우선순위】

① 丙火 (先丙解凍 - 필수)
  - 초춘의 寒氣를 녹이는 것이 최우선
  - "先"자가 있어 반드시 먼저 필요
  - 戊土가 발육하려면 暖이 필수
  - "天道有寒暖, 發育萬物"의 핵심
  → 寒氣가 심하면 丙火가 최우선

② 甲木 (次甲疏土 - 보조)
  - "次"자가 있어 丙火 다음 순위
  - 丙火로 暖氣가 돌면, 흙이 말라버릴 수 있음
  - 甲木이 흙을 소토(疏土)하여 활력을 줌
  - 흙을 풀어주어 만물이 자랄 수 있게 함
  → 暖氣가 돌면 甲木으로 소토하여 활력

【이 사주 적용】

현재 상태:
  - 丙火: 월간에 투출 + 강한 통근 ✅
  - 甲木: 시간에 투출 + 이중 본기 통근 ✅

조후 판단:
  - 丙火가 있어 寒氣를 녹임 ✅
  - 甲木이 있어 소토 작용 가능 ✅
  - 초춘의 조후 요구사항 충족

다만:
  - 甲木이 시간에 있어 위치상 약함
  - 그러나 이중 본기 통근으로 작용력 있음
  - 丙火가 주도하고 甲木이 보조하는 구조

```

###### 조건 1-2: 조습(燥濕) 확인

```
【적천수 원문】

> "地道有燥濕, 生成品彙, 人道得之, 不可偏也"
> (지도에 조습이 있어 품휘를 생성하니, 
>  인도가 이를 얻되 치우침이 없어야 한다)

【원문 해석】

地道有燥濕:
  - 지도(地道): 땅의 이치, 지지의 성질
  - 조습(燥濕): 건조함과 습함
  - 땅에는 건조함과 습함이 있음

生成品彙:
  - 품휘(品彙): 만물의 종류
  - 조습의 조화로 만물이 생성됨

人道得之, 不可偏也:
  - 인도(人道): 사람의 도리, 일간
  - 치우침이 없어야 함
  - 너무 건조하거나 습하지 않아야 함

【이 사주의 조습 분석】

戊土의 본래 성질:
  - 戊土(양토): 건조한 성질
  - 양토는 본래 건조함을 가짐

寅월의 특성:
  - 초춘으로 아직 습기가 약간 있음
  - 그러나 木氣가 왕성하여 土의 습기를 흡수
  - 전체적으로 건조한 편

사주 내 水:
  - 천간: 없음
  - 지지: 없음
  → 水가 없어 濕하지 않음

사주 내 火:
  - 丙火(월간) + 巳火(년지)
  → 火가 있어 燥해질 수 있음

판단:
  - 전체적으로 燥한 편
  - 그러나 寅월의 초춘 습기가 약간 있어 균형
  - "不可偏也" → 크게 치우치지 않음 ✅
  - 戊土의 본래 성질(건조)과 조화
  - 과도하게 건조하지 않아 만물 생성 가능

```

##### 조건 2: 일간이 한난을 어떻게 받는가

```
일간: 戊土

질문: 戊土가 寒한 환경에서 어떠한가?

土의 성질:
  - 차가우면 굳고 얼어붙음
  - 따뜻해야 부드럽고 만물을 기름
  - 그러나 과도하게 따뜻하면 말라버림
  - 적절한 소토(疏土)가 필요함

戊土 + 寒한 寅월:
  → 흙이 아직 차가운 상태
  → 따뜻함(暖)이 필요함
  → 暖이 돌면 소토(疏土)도 필요

【조후 우선순위 적용】:
  - 1순위: 丙火로 寒氣 해소 (필수)
  - 2순위: 甲木으로 소토하여 활력 (보조)
  → 丙火가 먼저 寒을 녹이고
  → 甲木이 흙을 풀어주어 발육 가능

판단: 暖을 얻어야 "發育萬物"할 수 있음
  → 丙火가 필수, 甲木이 보조

```

##### 조건 3: "不可過也" - 지나침 여부 확인

```
질문: 이 사주에서 한난이 지나친가?

寅월의 寒:
  - 극한(極寒)인 子월, 丑월이 아님
  - 초춘으로 한기가 있으나 극단적이지 않음

판단: 過하지 않음
  → 치명적 수준은 아님
  → 그러나 暖의 도움이 있으면 좋음

```

###### 조건 3-1: 暖의 "過" 여부 확인

```
질문: 丙火(暖)가 지나치지 않은가?

【적천수 원문 "不可過也" 적용】

"不可過也"의 의미:
  - 지나침이 없어야 함
  - 寒이 지나치면 안 되고, 暖도 지나치면 안 됨
  - 균형이 중요

【이 사주 暖의 과불급 분석】

丙火의 개수:
  - 천간: 丙火 1개
  - 지지: 巳火 1개 (丙火 본기)
  → 총 火가 2개

火의 과다 여부:
  - 火가 2개로 적절함
  - 과다하면:
    * 土가 말라버림 (晦土, 마른 흙)
    * 戊土가 타버려 만물을 기르기 어려움
    * "不可過也" 위반
  - 부족하면:
    * 寒氣 해소 못함
    * "發育萬物" 불가능

판단:
  - 暖이 適度함 ✅
  - "不可過也" 충족 ✅
  - 丙火가 寒을 녹이되, 과도하지 않음
  - 戊土가 말라버리지 않는 적절한 수준
  - "人道得之, 不可過也" 완벽 충족

```

##### 조건 4: 暖을 주는 것이 사주에 있는가

```
질문: 사주 내에 따뜻함(暖)을 주는 것이 있는가?

暖을 주는 오행: 火

사주 내 火:
  - 천간: 丙火 (월간) ✅
  - 지지: 巳火 (년지) ✅

丙火의 상태:
  - 월간에 투출 (드러나 작용함)
  - 巳에 본기 통근 (뿌리 있음)
  - 寅에 중기 통근 (추가 뿌리)

판단: ✅ 暖이 갖춰짐
  → "人道得之" 충족

```

##### 조건 5: 조후용신과 격국용신의 구분

```
【중요】 적천수에서의 용신 구분:

적천수에서는 조후를 중시하나,
조후용신이 곧 격국용신은 아님

① 조후용신 (調候用神):
  - 천도의 한난(寒暖)을 조화시키는 것
  - 계절의 기후적 요구사항을 충족시키는 것
  - "급한 것이 해결된 것"
  - 예: 寅월 戊土에게 丙火는 조후용신

② 격국용신 (格局用神):
  - 사주 전체의 구조를 조화롭게 만드는 것
  - 희기신(喜忌神)의 작용을 조절하는 것
  - "최종 용신"
  - 조후, 쇠왕, 종격, 희기, 중화 등을 종합 판단

【이 사주 적용】

조후용신:
  - 丙火: 寅월의 寒을 녹여 暖을 줌
  - 조후 요구사항 충족 ✅
  → "급한 것(寒氣)이 해결됨"

격국용신 (아직 확정 전):
  - Step 1~6을 모두 거쳐야 확정
  - 조후가 충족되었다고 해서
    곧바로 격국용신이 되는 것은 아님
  - 다른 단계(쇠왕, 종격, 희기, 중화)도 확인 필요

【적천수 판단 흐름】

Step 2 (조후):
  → 조후용신 확인: 丙火 ✅
  → "급한 것이 해결됨"
  → 그러나 이것이 "최종 용신"은 아님

다음 단계:
  → 쇠왕, 종격, 희기, 중화 등을 확인하여
  → 최종 격국용신 확정

```

## Step 3: 쇠왕 심사 (衰旺 審査)

> 핵심 질문: "일간의 쇠왕의 진기(眞機)는 어떠한가?"

### 3-1. 적천수 원문

**원문:**

> "能知衰旺之真機, 其於三命之奧, 思過半矣"
(쇠왕의 진기를 알면 삼명의 오묘함을 절반 이상 안 것이다)
> 

---

### 3-2. 원문 해석

### "衰旺"

```
衰 = 쇠함, 약함
旺 = 왕성함, 강함

해석:
  일간의 힘이 쇠한가, 왕한가
  → 단순한 강약이 아닌 실질적 상태

```

### "真機"

```
真 = 참된, 진짜의
機 = 기틀, 핵심 작용

해석:
  겉으로 보이는 것이 아닌 참된 기틀
  → 오행 개수가 아닌 실제 작용하는 힘
  → 표면이 아닌 본질

```

### "三命之奧, 思過半矣"

```
三命 = 명리학 전체
奧 = 오묘함, 깊은 이치
思過半 = 절반 이상을 생각함

해석:
  쇠왕의 진기를 알면 명리의 절반 이상을 아는 것
  → 그만큼 쇠왕 판단이 핵심이라는 뜻
  → 그러나 "진기"를 알아야 함 (겉모습 아님)

```

### 3-3. 이 사주에 적용

#### 3-3-1. 쇠왕 분석

##### 조건 1: 월령에서의 쇠왕

```
질문: 일간이 월령에서 왕한가, 쇠한가?

일간: 戊土
월령: 寅木 (木이 사령)

월령과의 관계:
  - 寅월은 木이 왕성한 때
  - 木은 土를 극함 (木克土)
  - 일간이 월령에서 극을 받음

판단: 衰 (월령에서 쇠함)

```

###### 조건 1-1: 신강/신약 판단의 점수화 접근

```
【현대 명리학 4가지 기준 - 점수 배점】

적천수 "眞機"를 파악하기 위해
현대 명리학의 점수화 기준을 적용:

각 항목별 만점:
  - 득령: 4점 만점
    → 당령(사령): 4점
    → 생왕: 3점
    → 휴수: 1점
    → 사절: 0점
  
  - 득지: 5점 만점
    → 본기 통근 1개당: 1.5점
    → 중기 통근 1개당: 1점
    → 여기 통근 1개당: 0.5점
    → 일지 본기 통근: +0.5점 가산
    → (본기 통근 2개 + 일지 가산 시 최대 5점)
  
  - 득세: 2점 만점
    → 비겁 1개: 1점
    → 비겁 2개 이상: 2점
  
  - 득조: 3점 만점
    → 생조 오행 유력: 3점
    → 생조 오행 있으나 약함: 2점
    → 생조 오행 미약: 1점

─────────────────────────────────────

【이 사주 적용】

① 得令 (득령) - 월령에서의 상태
   - 寅월에 戊土 → 휴수(休囚)
   - 월령에서 실령하여 득령 못함 ❌
   - 점수: 0점 (득령 못함)

② 得地 (득지) - 지지에 통근 여부
   - 戌(일지)에 본기 통근 ✅ → 1.5점
   - 일지 본기 통근 가산 → +0.5점
   - 巳(년지)에 본기 통근 ✅ → 1.5점
   - 寅(월지, 시지)에 여기 통근 ✅ → 0.5점 × 2 = 1점
   - 합계: 5점 (득지 매우 강함)

③ 得勢 (득세) - 같은 오행의 세력
   - 己土(년간) 있음 ✅
   - 같은 土로 힘을 보탬
   - 다만 1개뿐이라 강하지 않음
   - 점수: 1점 (약간의 세력)

④ 得助 (득조) - 생조하는 오행
   - 丙火(월간)가 火生土로 생조 ✅
   - 월간 투출 + 강한 통근
   - 매우 유력한 생조
   - 점수: 3점 (득조함, 매우 유력)

【점수 종합】

득령: 0점
득지: 5점
득세: 1점
득조: 3점
─────────────────
총점: 9점 / 13점 (만점 기준)

【일간 강약 등급 구분 기준】

등급 구분 (점수 기반):
- 극약(極弱): 0~2점
  → 무너짐, 종격 고려 필요

- 편약(偏弱): 3~5점
  → 약하나 버틸 수 있음
  → 眞機가 있으면 일반격 가능

- 중화(中和): 6~8점
  → 균형 잡힌 상태
  → 가장 이상적인 상태

- 편강(偏强): 9~10점
  → 강한 상태
  → 설기(泄氣) 필요할 수 있음

- 극강(極强): 11~12점
  → 매우 강한 상태
  → 설기 또는 극파 필요

【이 사주 판단】

득령: 0점 ❌
득지: 5점 ✅
득세: 1점 △
득조: 3점 ✅
─────────────────
총점: 9점

→ 등급: 편강(偏强) 또는 중화 상위
  → 득지가 매우 강함 (5점, 본기 통근 2개)
  → 득조도 강함 (3점)
  → 다만 득령이 없어 완전한 편강은 아님
  → 편강 하위 또는 중화 상위 수준

```

#### 3-3-2. 진기(眞機) 분석

##### 조건 2: 진기(眞機) 파악 - 겉과 속

```
질문: 겉으로 쇠해 보이나, 진기는 어떠한가?

"眞機"를 보려면:
  - 겉모습(월령 실령)만 보지 않음
  - 실제로 일간이 작용할 힘이 있는가를 봄

【眞機 판단의 구체적 기준】

적천수 "能知衰旺之眞機"에서
眞機를 판단하는 구체적 기준:

① 통근 여부 (通根)
  - 지지에 일간의 뿌리가 있는가?
  - 본기(本氣) > 중기(中氣) > 여기(餘氣)
  - 일지(坐下) 통근이 가장 중요
  → 이 사주: 戌(일지) 본기 통근 ✅

② 천투지장 (天透地藏)
  - 천간에 투출하고 지지에 뿌리가 있는가?
  - 투출한 오행이 지지에 통근하면 강함
  → 이 사주: 丙火 월간 투출 + 巳寅 통근 ✅

③ 유근무근 (有根無根)
  - 유근: 뿌리가 있어 안정됨
  - 무근: 뿌리가 없어 불안정함
  → 이 사주: 유근 (戌 본기 + 巳寅 여기) ✅

④ 생조 관계
  - 일간을 생하는 오행이 있는가?
  - 생조하는 오행이 유력한가?
  → 이 사주: 丙火 생조 + 유력함 ✅

⑤ 극파 관계
  - 일간을 극하는 오행이 제어되는가?
  - 극하는 오행이 전환되는가?
  → 이 사주: 甲木이 丙火로 전환됨 ✅

【眞機 종합 체크리스트】

□ 통근 여부: ✅ 일지 본기 통근
□ 천투지장: ✅ 丙火 투출 + 통근
□ 유근무근: ✅ 유근 (뿌리 있음)
□ 생조 관계: ✅ 丙火 유력한 생조
□ 극파 관계: ✅ 甲木 전환됨

결론:
  → 眞機가 있음
  → 겉으로 쇠하나 실제로 무너지지 않음

```

##### 조건 3: 일간의 뿌리(根) 확인

```
질문: 戊土가 지지에 뿌리가 있는가?

【통근 강도 분석】

지지 검토:
  - 년지 巳: 지장간 庚(여기)·丙(중기)·戊(본기) → 戊 있음 (본기) ✅
  - 월지 寅: 지장간 戊·丙·甲 → 戊 있음 (여기) ✅
  - 일지 戌: 지장간 辛·丁·戊 → 戊 있음 (본기) ✅
  - 시지 寅: 지장간 戊·丙·甲 → 戊 있음 (여기) ✅

통근 강도:
  - 본기(本氣): 가장 강함 → 戌(일지) ✅
  - 중기(中氣): 중간 강도 → 없음
  - 여기(餘氣): 약함 → 巳, 寅×2

【일지(坐下) 통근의 중요성】

특히 일지(坐下):
  - 戌土에 戊土 본기
  - 일간이 앉은 자리에 강한 뿌리
  - 적천수에서 가장 중시하는 통근
  - "眞機"의 핵심 요소

【통근 종합 평가】

본기 통근: 1개 (戌) ✅
중기 통근: 0개
여기 통근: 3개 (巳, 寅×2) ✅

판단: 根이 있음
  → 일지 본기 통근으로 강한 뿌리
  → 겉으로 쇠하나 뿌리가 있어 무너지지 않음
  → 眞機의 핵심 조건 충족

```

##### 조건 4: 일간을 돕는 것 확인

```
질문: 천간에서 일간을 돕는 것이 있는가?

【생조와 비조 분석】

천간 검토:
  - 년간 己土: 같은 土로 힘을 보탬 (비조) ✅
  - 월간 丙火: 火生土로 일간을 생함 (생조) ✅
  - 시간 甲木: 木克土로 일간을 극함 (극) ❌

【생조 강도 분석】

丙火 (생조):
  - 위치: 월간 (월령에서 투출)
  - 통근: 巳(본기) + 寅(중기)
  - 작용: 火生土로 일간을 생함
  - 강도: 매우 강함 ✅

己土 (비조):
  - 위치: 년간
  - 통근: 없음 (무근) ⚠️
    → 巳: 己土 없음 (戊庚丙)
    → 寅: 己土 없음 (戊丙甲)
    → 戌: 己土 없음 (辛丁戊)
  - 작용: 같은 土로 심리적 도움
  - 강도: 약함 (무근이라 실질적 도움 제한적)

【돕는 것 vs 해치는 것】

돕는 것:
  - 丙火: 매우 강한 생조 ✅
  - 己土: 보조적 비조 ✅

해치는 것:
  - 甲木: 강하나 丙火로 전환됨 ⚠️

판단: 돕는 것이 있음
  → 丙火가 매우 유력하게 생조함
  → 己土가 보조적으로 힘을 보탬
  → 眞機의 핵심: 생조가 유력함

```

##### 조건 5: 해치는 것의 상태 확인

```
질문: 일간을 해치는 甲木은 어떤 상태인가?

甲木의 힘:
  - 시간에 투출
  - 寅寅에 본기 통근 (강한 뿌리)
  - 월령을 얻어 왕성함

甲木의 작용:
  - 직접 戊土를 극하는가?
  - 아니면 다른 경로가 있는가?

확인:
  - 甲木 → 丙火 → 戊土 경로 존재
  - 甲이 丙을 생하고 (木生火)
  - 丙이 戊를 생함 (火生土)
  - 중간에 丙火가 있어 전환됨

판단: 해치는 것이 전환됨
  → 甲木이 강하나 丙火가 받아줌
  → 직접적 극이 완화됨

```

##### 조건 6: 형충회합이 통근에 미치는 영향

```
【이 사주의 지지 관계】

寅-寅 (월지-시지):
  - 관계: 자형(自刑) 또는 병렬
  - 영향: 木氣 증폭 → 土 극 강화 가능성
  - 통근 영향: 직접적 영향 적음
    → 戊土는 寅에 여기 통근만 있음
    → 자형으로 인한 통근 손상 없음

寅-巳 (월지-년지):
  - 관계: 형(刑) 또는 해(害) - 학파에 따라
  - 영향: 巳가 손상되면 丙火 통근 약화 가능
  - 통근 영향:
    → 戊土는 巳에 본기 통근
    → 巳가 손상되면 戊土 통근 약화 가능 ⚠️
  - 판단: 현재 巳가 건재하여 큰 영향 없음
    → 다만 형해 관계로 인한 잠재적 약화 가능성

寅-戌 (월지-일지):
  - 관계: 寅午戌 삼합 중 반합 (午 없음)
  - 영향: 불성립, 영향 없음
  - 통근 영향: 없음
    → 戊土는 戌에 본기 통근
    → 반합 불성립으로 통근 영향 없음

【통근 영향 종합】

戊土의 통근 상태:
  - 戌(일지): 본기 통근 ✅
  - 巳(년지): 본기 통근 ✅
  - 寅(월지, 시지): 여기 통근 ✅

형충회합 영향:
  - 寅-寅 자형: 통근 영향 없음 ✅
  - 寅-巳 형해: 잠재적 약화 가능성 있으나 현재 건재 ✅
  - 寅-戌 반합: 영향 없음 ✅

판단:
  → 형충회합으로 인한 통근 손상 없음 ✅
  → 戊土의 통근 유지됨 ✅
  → 특히 일지 본기 통근(戌)이 안정적
  → 진기(眞機)의 핵심인 통근이 보존됨

```

### 3-4. 진기(眞機) 종합 판정

```
적천수 "眞機" 기준:

【겉모습 (表面)】
  - 월령 실령 → 쇠해 보임
  - 득령 못함 ❌

【진기(眞機) - 구체적 기준 적용】

① 통근 여부:
  - 일지 본기 통근 ✅
  - 년지, 월지, 시지 여기 통근 ✅
  → 유근(有根) 확정

② 천투지장:
  - 丙火 월간 투출 + 巳寅 통근 ✅
  - 생조 오행이 유력함 ✅

③ 생조 관계:
  - 丙火 매우 유력한 생조 ✅
  - 己土 보조적 비조 ✅

④ 극파 관계:
  - 甲木이 丙火로 전환됨 ✅
  - 직접적 극이 완화됨 ✅

【眞機 종합 평가】

체크리스트:
  □ 통근: ✅ 일지 본기
  □ 천투지장: ✅ 丙火 유력
  □ 유근무근: ✅ 유근 (뿌리 있음)
  □ 생조: ✅ 丙火 매우 유력
  □ 극파: ✅ 전환됨

결론:
  "衰하나 眞機가 있음"
  → 겉으로 쇠하나 실제로 무너지지 않는 상태

## Step 4: 종격 심사 (從格 審査)

> 핵심 질문: "일간이 종(從)할 것인가, 종하지 않을 것인가?"
> 

---

### 4-1. 적천수 원문

**원문 1:**

> "從得真者只論從, 從神又有吉和凶"
(종이 참됨을 얻으면 오직 종으로 논하고, 종신에도 길흉이 있다)
> 

**원문 2:**

> "假從亦可例言從, 見從不從反為凶"
(가종도 종으로 논할 수 있으나, 종인 듯 종이 아니면 도리어 흉하다)
> 

---

### 4-2. 원문 해석

### "從得真者只論從"

```
從 = 따름, 복종함
得真 = 참됨을 얻음
只論從 = 오직 종으로 논함

해석:
  종격이 참되게 성립하면
  → 다른 것 볼 필요 없이 종으로만 판단
  → 종신(따르는 대상)이 용신이 됨

```

### "從神又有吉和凶"

```
從神 = 종하는 대상 (따르는 신)
吉和凶 = 길함과 흉함

해석:
  종격이 성립해도 종신의 상태에 따라 길흉이 갈림
  → 종신이 온전하면 길
  → 종신이 파손되면 흉

```

### "假從亦可例言從"

```
假從 = 가짜 종격
亦可 = 또한 가능하다
例言從 = 종으로 논함

해석:
  완전한 종격이 아니어도 종으로 볼 수 있는 경우
  → 거의 종격에 가까운 상태

```

### "見從不從反為凶"

```
見從不從 = 종인 듯 종이 아님
反為凶 = 도리어 흉함

해석:
  종격처럼 보이나 실제로 종이 아닌 경우
  → 억지로 종으로 보면 흉함
  → 진종(眞從)과 가종(假從)을 구별해야 함

```

### 4-3. 이 사주에 적용

#### 4-3-1. 종격 성립 여부 분석

##### 조건 1: 종(從)할 상황인가?

```
질문: 일간이 종해야 할 만큼 무력한가?

종격의 전제:
  - 일간이 극히 약하여 스스로 버틸 수 없음
  - 한 세력이 압도적으로 강함
  - 일간을 돕는 것이 없거나 무력함

Step 3 결과 확인:
  - 일간 戊土: 衰하나 眞機 있음
  - 일지 戌에 본기 뿌리 있음
  - 丙火, 己土의 생조 있음

판단: ❌ 종할 상황 아님
  → 일간이 무력하지 않음
  → 뿌리와 생조가 있어 버틸 수 있음

```

【Step 3 결과 연계】

종격 성립 기준 (점수화):
  - 극약(極弱): 0~2점 → 진종 고려
  - 편약(偏弱): 3~5점 → 가종 가능성
  - 중화(中和) 이상: 6점 이상 → 종격 불성립

이 사주 Step 3 결과:
  - 총점: 9점 (편강 하위 또는 중화 상위)
  - 6점 이상 → 종격 불성립 기준 충족

결론: 정량적으로도 종격 불성립 확정
  → 점수화 접근으로도 종격이 아님을 확인
  → Step 3의 정량적 판단과 일치

```

###### 조건 1-1: 종격 유형별 검토

```
【종격 유형별 성립 조건】

종격은 종하는 대상에 따라 구분됨:
- 종살격(從殺格): 관살(官殺)을 종함
- 종재격(從財格): 재성(財星)을 종함
- 종아격(從兒格): 식상(食傷)을 종함
- 종세격(從勢格): 여러 오행의 세력을 종함

【각 종격 유형별 검토】

① 종살격(從殺格) 검토
  - 조건: 관살이 압도적으로 강함
  - 일간이 극히 약하여 관살을 종함
  
  이 사주:
  - 관살: 甲木 (시간) + 寅寅 (월지, 시지)
  - 甲木의 세력:
    * 시간 투출 ✅
    * 寅寅 본기 통근 (이중 통근) ✅
    * 월령 득세 ✅
    → 관살이 강함
  
  그러나:
  - 일간 戊土가 戌에 본기 통근 ✅
  - 丙火가 유력하게 생조함 ✅
  - 己土가 힘을 보탬 ✅
  → 일간이 무력하지 않음
  → 종살격 불성립 ❌

② 종재격(從財格) 검토
  - 조건: 재성이 압도적으로 강함
  - 일간이 극히 약하여 재성을 종함
  
  이 사주:
  - 재성: 없음 (水가 없음)
  → 종재격 불성립 ❌

③ 종아격(從兒格) 검토
  - 조건: 식상이 압도적으로 강함
  - 일간이 극히 약하여 식상을 종함
  
  이 사주:
  - 식상: 庚辛金 (없음)
  → 종아격 불성립 ❌

④ 종세격(從勢格) 검토
  - 조건: 여러 오행의 세력이 압도적
  - 일간이 극히 약하여 세력을 종함
  
  이 사주:
  - 木이 강함 (월령 + 이중 통근)
  - 火가 있음 (월간 투출 + 통근)
  - 土가 있음 (일지 본기 통근)
  → 여러 오행이 있으나
  → 일간이 戌에 본기 통근하여 무력하지 않음
  → 종세격 불성립 ❌

⑤ 종강격(從强格) / 종왕격(從旺格) 검토
  - 조건: 일간이 극히 강하여 억제 불가
  - 비겁이 압도적으로 많음
  - 일간을 돕는 것을 용신으로 삼음
  
  이 사주:
  - 戊土: Step 3 기준 7점 (중화 하위)
  - 비겁: 己土 1개 (년간)
  - 극강이 아니며 비겁 압도적이지 않음
  → 종강격/종왕격 불성립 ❌

【종격 유형별 종합 판단】

모든 종격 유형 불성립:
  - 종살격: 관살 강하나 일간이 버팀 ❌
  - 종재격: 재성 없음 ❌
  - 종아격: 식상 없음 ❌
  - 종세격: 일간이 무력하지 않음 ❌
  - 종강격/종왕격: 일간이 극강이 아니며 비겁 압도적이지 않음 ❌

결론: 종격 불성립

```

#### 4-3-2. 진종/가종 구분

##### 조건 2: "從得真者" 체크 - 진종(眞從)과 가종(假從)의 구분

```
【적천수 원문의 의미】

"從得真者只論從, 從神又有吉和凶"
→ 종이 참됨을 얻으면(眞從) 오직 종으로만 논함
→ 종신에도 길흉이 있음

"假從亦可例言從, 見從不從反為凶"
→ 가종도 종으로 논할 수 있으나
→ 종인 듯 종이 아니면(見從不從) 도리어 흉함

【진종(眞從)과 가종(假從)의 구분】

진종(眞從)의 조건:
  ① 일간이 뿌리가 전무하거나 극히 미약
  ② 일간을 돕는 인비가 없거나 무력
  ③ 한 오행이 압도적으로 강함
  ④ 종신이 온전함 (파손되지 않음)

가종(假從)의 조건:
  - 진종은 아니나 거의 종에 가까움
  - 일간의 뿌리가 아주 미약함
  - 인비가 있으나 힘이 약함
  - 종신이 약간 파손되거나 불완전함

【진종과 가종의 해석 차이】

진종(眞從):
  - 완전한 종격
  - 종신이 용신이 됨
  - 종신을 돕는 것이 길함
  - 종신을 해치는 것이 흉함

가종(假從):
  - 불완전한 종격
  - 종신이 용신이 되나 약함
  - 종신을 돕는 것이 길하나 완전하지 않음
  - 종신을 해치는 것이 흉함

【이 사주 체크】

진종(眞從) 조건:
  ① 뿌리: 戌土 본기 + 巳寅寅에 여기 → ❌ 뿌리 있음
  ② 인비: 丙火 투출 + 통근, 己土 투출 → ❌ 유력한 인비 있음
  ③ 압도: 木이 강하나 火土도 버팀 → ❌ 완전 압도 아님
  ④ 종신: 甲木 강하나 丙火가 전환함 → ❌ 완전 압도 아님

가종(假從) 조건:
  - 일지 戌에 본기 통근 → 미약하지 않음 ❌
  - 丙火가 월간 투출 + 강한 통근 → 유력함 ❌

판단: ❌ 진종 불성립, ❌ 가종도 불성립
  → 세 조건 모두 충족 안 됨
  → 일간이 무력하지 않아 종격 자체가 성립 안 됨

```

##### 조건 3: "假從亦可例言從" 체크

```
질문: 가종(假從)으로라도 볼 수 있는가?

가종의 조건:
  - 진종은 아니나 거의 종에 가까움
  - 일간의 뿌리가 아주 미약함
  - 인비가 있으나 힘이 약함

이 사주:
  - 일지 戌에 본기 통근 → 미약하지 않음
  - 丙火가 월간 투출 + 강한 통근 → 유력함

판단: ❌ 가종도 불성립
  → 일간의 뿌리와 인비가 유력함
  → (조건 2에서 이미 검토 완료)

```

#### 4-3-3. 화격(化格) 검토

##### 조건 3-1: 화격(化格) 검토

```
【화격(化格)의 개념】

화격(化格) = 화기격(化氣格)
- 일간이 다른 천간과 합(合)하여 화기(化氣)로 변함
- 합한 오행으로 화함
- 종격과는 다른 특수격

【일간 戊土의 합(合)】

戊土가 합하는 천간:
  - 癸水와 합하여 火로 화함 (戊癸合化火)
  - 합화 조건:
    ① 천간에 癸水가 있어야 함
    ② 월령이 火旺한 때이거나
    ③ 火가 강하게 통근해야 함

【이 사주 화격 검토】

천간 확인:
  - 년간: 己土
  - 월간: 丙火
  - 일간: 戊土
  - 시간: 甲木
  → 癸水 없음 ❌

화격 성립 조건:
  ① 癸水 존재: ❌ 없음
  ② 월령 火旺: ❌ 寅월 (木旺)
  ③ 火 강한 통근: ✅ 丙火 + 巳寅 통근

판단: ❌ 화격 불성립
  → 癸水가 없어 합 자체가 성립 안 됨
  → 화격 검토 단계는 거쳤으나 불성립

【화격과 종격의 관계】

화격이 성립하면:
  - 일간이 火로 화함
  - 火를 용신으로 삼음
  - 종격과는 별개의 특수격

이 사주:
  - 화격 불성립
  - 종격도 불성립
  → 일반격으로 판단

```

###### 조건 3-2: 甲己合 검토

```
【甲己合의 존재】

사주 내 천간:
  - 시간 甲木
  - 년간 己土
  → 甲己合 존재 (土로 화함 가능)

【합(合)의 성립 여부】

합의 조건:
  - 인접 여부: 시간과 년간 → 원격(떨어져 있음) ⚠️
  - 중간 방해: 丙(월간), 戊(일간)이 가로막음
  
판단: 합은 있으나 힘이 약함 (원격합)
  → 시간과 년간 사이에 월간, 일간이 끼어있음
  → 직접적인 합의 힘이 약함

【화(化)의 성립 여부】

甲己合化土 조건:
  ① 월령이 土旺해야 함
     → 寅월은 木旺 ❌
  ② 土가 사주에 강해야 함
     → 戊己土 있으나 압도적이지 않음 △
  ③ 화기를 파하는 것이 없어야 함
     → 木이 강해 土를 극함 ❌

판단: 甲己合은 있으나 化는 불성립
  → 원격합 + 木旺 월령 → 화 조건 불충족
  → 일반적 작용으로 봄 (합으로 인한 牽制 정도)
  → 甲木과 己土가 서로를 약간 억제하는 정도

【甲己合이 종격 판단에 미치는 영향】

甲己合의 의미:
  - 甲木(편관)과 己土(비겁)가 합함
  - 甲木의 극(克) 작용이 약화됨
  - 己土의 비조 작용도 약화됨

종격 판단에 영향:
  - 甲木이 약화되어 종살격 가능성 더욱 낮아짐
  - 己土가 약화되어 비조 작용 제한적
  - 다만 원격합이라 영향은 제한적
  → 종격 불성립 판단에 변화 없음

```

### 5-2. 원문 해석

### "何知其人吉, 喜神爲輔持"

```
何知 = 어찌 아는가
吉 = 길함
喜神 = 기뻐하는 신, 이로운 것
輔持 = 보좌하여 지탱함

해석:
  사람이 길한 것은 희신이 보좌하기 때문
  → 희신이 있고, 그것이 일간을 돕고 있어야 함
  → 희신이 있어도 작용하지 못하면 소용없음

```

### "何知其人凶, 忌神輾轉攻"

```
凶 = 흉함
忌神 = 꺼리는 신, 해로운 것
輾轉 = 이리저리 구름, 번갈아
攻 = 공격함

해석:
  사람이 흉한 것은 기신이 번갈아 공격하기 때문
  → 기신이 제어 없이 일간을 해침
  → 기신이 있어도 제어되면 흉하지 않음

```

### "性定元氣厚"

```
性定 = 성품이 안정됨
元氣厚 = 원기가 두터움

해석:
  사주의 기운이 안정되고 두터우면 좋음
  → 한쪽으로 치우치지 않음
  → 희신이 보좌하고 기신이 제어된 상태

```

輔持의 조건:
  ① 천간에 드러나 작용함
  ② 지지에 뿌리가 있어 안정됨
  ③ 극파되지 않아 온전함
  ④ 일간에게 실질적 도움을 줌

丙火 체크:
  ① 투출: 월간에 투출 ✅
  ② 통근: 巳(본기), 寅(중기) ✅
  ③ 극파: 壬癸水 없음 ✅
  ④ 작용: 火生土로 일간 생조 ✅

판단: ✅ 丙火가 輔持하고 있음

己土 체크:
  ① 투출: 년간에 투출 ✅
  ② 통근: 없음 (무근) ⚠️
     → 巳: 己土 없음 (戊庚丙)
     → 寅: 己土 없음 (戊丙甲)
     → 戌: 己土 없음 (辛丁戊)
  ③ 극파: 직접 극하는 것 없음 ✅
  ④ 작용: 같은 土로 심리적 도움 △

판단: △ 己土도 輔持하나 무근으로 약함
  → 보조적 희신 (핵심 희신은 丙火)

  → 기신의 힘은 강함

甲木의 작용 경로:
  - 경로 A: 甲 → 戊 (직접 극)
  - 경로 B: 甲 → 丙 → 戊 (전환)

현재 상태:
  - 丙火가 월간에 있음 (甲과 戊 사이)
  - 甲木이 丙火를 생함 (木生火)

  → 경로 B가 작동함
  → 甲의 기운이 丙을 통해 전환됨

판단: ❌ 輾轉攻 아님
  → 기신(甲木)이 강하나
  → 희신(丙火)이 중간에서 받아 전환함
  → 직접 공격이 아닌 간접 생조로 바뀜

#### 5-3-4. 형충회합 영향 분석

##### 조건 5: 형충회합(刑沖會合)에 따른 변화 분석

```
【형충회합의 개념】

지지 간의 특수한 관계:
- 합(合): 화합, 결합
- 충(沖): 충돌, 대립
- 형(刑): 형벌, 상극
- 회(會): 모임, 집합
- 해(害): 해침, 손상

이런 관계들이 오행의 작용에 영향을 줌
→ 희기신의 작용력이 변화할 수 있음

【이 사주의 지지 관계 분석】

사주 지지: 巳(년지) 寅(월지) 戌(일지) 寅(시지)

① 寅戌 반합(半合) 검토
  - 寅午戌 삼합(三合)의 일부
  - 午가 없으므로 반합(半合)
  - 寅戌 반합 → 火局을 형성하려 함
  
  이 사주:
  - 寅(월지, 시지) + 戌(일지)
  - 午가 없어 완전한 삼합은 아님
  - 그러나 丙火가 월간에 있어 火氣가 있음
  
  영향:
  - 寅戌 반합으로 火氣가 모임
  - 희신(丙火)의 작용력이 강화됨 ✅
  - 火局을 형성하려는 기운

② 寅巳 형(刑) 또는 상해(相害) 검토
  - 寅巳 상형(相刑): 서로 형벌함
  - 寅巳 상해(相害): 서로 해침
  
  이 사주:
  - 寅(월지, 시지) + 巳(년지)
  - 寅巳 상형 또는 상해 관계
  
  영향 분석:
  - 寅(木)과 巳(火)의 충돌
  - 巳는 丙火의 본기 통근지
  - 寅은 甲木의 본기 통근지
  - 寅巳 형해로 인해:
    * 丙火의 뿌리(巳)가 약화될 수 있음 ⚠️
    * 甲木의 뿌리(寅)도 약화될 수 있음 ⚠️
    * 그러나 丙火가 월간에 투출하여 작용력 유지 ✅
  
  종합 판단:
  - 寅巳 형해가 있으나
  - 丙火가 월간 투출로 작용력 유지
  - 寅戌 반합으로 火氣가 모여 상쇄됨
  → 희신(丙火)의 작용력은 유지됨

③ 형충회합이 희기신에 미치는 영향

【희신(丙火)에 미치는 영향】

긍정적 영향:
  - 寅戌 반합으로 火氣 모임 ✅
  - 火局 형성 기운으로 작용력 강화 ✅

부정적 영향:
  - 寅巳 형해로 巳(본기 통근지) 약화 ⚠️
  - 다만 월간 투출로 작용력 유지 ✅

종합:
  - 긍정적 영향이 부정적 영향보다 큼
  - 희신(丙火)의 작용력은 유지되거나 강화됨 ✅

【기신(甲木)에 미치는 영향】

긍정적 영향:
  - 없음

부정적 영향:
  - 寅巳 형해로 寅(본기 통근지) 약화 ⚠️
  - 寅戌 반합으로 火氣가 모여 木氣 약화 ⚠️

종합:
  - 기신(甲木)의 작용력이 약화됨 ✅
  - 이는 희신(丙火)에게 유리함
  - 기신이 약해지면 희신이 더 유리함

【형충회합 종합 평가】

寅戌 반합:
  - 火氣 모임 → 희신(丙火) 강화 ✅

寅巳 형해:
  - 丙火와 甲木 모두 약화되나
  - 丙火는 월간 투출로 유지 ✅
  - 甲木은 약화되어 유리함 ✅

결론:
  - 형충회합이 희신(丙火)에게 유리하게 작용
  - 기신(甲木)의 작용력이 약화됨
  → 희기신의 작용력 차이가 더 벌어짐
  → 희신이 더 유리한 구조

```

희신(喜神):
  - 丙火: "輔持"함 ✅
    → 월간 투출, 강한 통근
    → 일간을 생조하고 기신을 전환함

  - 己土: "輔持"함 △
    → 년간 투출, 무근으로 약함

    → "輾轉攻" 하지 못함
    → 丙火가 전환하여 해침이 완화됨
    → 寅巳 형해로 뿌리 약화 ⚠️
    → 寅戌 반합으로 火氣 모여 약화됨 ⚠️

  - 寅木: 甲의 뿌리
    → 기신의 근원이나
    → 丙火도 寅에서 중기 통근하여 공유함
    → 寅巳 형해로 약화됨 ⚠️

【희기신 우선순위】

희신 순위:
  1순위: 丙火 (핵심 희신)
    → 조후용신 + 통관용신 + 병약용신

  2순위: 土 (戊己土) (보조 희신)
    → 丙火가 생하는 대상
    → 己土는 무근으로 약함

  1순위: 甲木 (핵심 기신)
    → 편관으로 일간을 극함

  2순위: 寅木 (기신의 뿌리)
    → 甲木과 丙火 모두 통근

```

### 6-2. 원문 해석

### "中和之氣, 其福獨鍾"

```
中和 = 치우치지 않고 고르게 조화됨
之氣 = 의 기운
其福 = 그 복
獨鍾 = 홀로 모임

해석:
  중화의 기운을 이루면 복이 모임
  → 한쪽으로 치우치지 않음
  → 기운의 흐름이 막힘 없이 통함
  → 희신이 작용하고 기신이 제어된 상태

```

### "旺相休囚, 辨其虛實"

```
旺相休囚 = 왕, 상, 휴, 수 (오행의 네 가지 상태)
辨 = 분별함
虛實 = 허함과 실함

해석:
  오행의 상태를 분별하여 허실을 봄
  → 겉으로 왕해 보여도 속이 허할 수 있음
  → 겉으로 쇠해 보여도 속이 실할 수 있음
  → 진기(眞機)를 봐야 함

```

### "源浚流淸, 斯眞貴矣"

```
源浚 = 근원이 깊음
流淸 = 흐름이 맑음
斯 = 이것이
眞貴 = 참으로 귀함

해석:
  근원이 깊고 흐름이 맑으면 참으로 귀함
  → 오행의 흐름이 막힘 없이 연결됨
  → 생극의 관계가 순조로움

```

### 6-3. 이 사주에 적용

#### 6-3-1. 중화 조건 분석

##### 조건 1: "中和之氣" 체크 - 치우침 여부

```
질문: 이 사주가 한쪽으로 치우쳐 있는가?

오행 분포 확인:
  - 木: 甲(천간) + 寅寅(지지) → 강함
  - 火: 丙(천간) + 巳(지지) → 있음
  - 土: 戊己(천간) + 戌(지지) → 있음
  - 金: 지장간에만 → 약함
  - 水: 없음 → 결여

치우침 여부:
  - 木이 가장 강함 (월령 + 다수)
  - 그러나 火와 土도 버티고 있음
  - 완전한 한쪽 치우침은 아님

판단: △ 木이 강하나 극단적 치우침은 아님

```

##### 조건 2: "旺相休囚, 辨其虛實" 체크

```
질문: 각 오행의 허실(虛實)은 어떠한가?

木의 허실:
  - 겉: 월령 득세, 寅寅 통근 → 旺해 보임
  - 속: 丙火가 木氣를 빼감 → 완전 實은 아님

火의 허실:
  - 겉: 천간 1개, 지지 1개 → 적어 보임
  - 속: 월간 투출 + 巳寅 통근 + 木의 생조 → 實함

土의 허실:
  - 겉: 월령 실령 → 衰해 보임
  - 속: 일지 본기 통근 + 火의 생조 → 虛하지 않음

판단:
  - 木: 旺하나 火에 설기되어 완전 實 아님
  - 火: 적어 보이나 實함
  - 土: 쇠해 보이나 虛하지 않음 (眞機 있음)

```

##### 조건 3: "源浚流淸" 체크 - 흐름의 연결

```
질문: 오행의 흐름이 막힘 없이 연결되는가?

흐름 확인:

木 → 火 → 土 경로:
  甲木(시간) → 丙火(월간) → 戊土(일간)
  寅木(월지) → 巳火(년지) → 戌土(일지)

  - 木이 火를 생함 (木生火) ✅
  - 火가 土를 생함 (火生土) ✅
  - 상생으로 연결됨 ✅

막힘 여부:
  - 木과 土 사이에 火가 있음
  - 木이 土를 직접 극하지 않음
  - 火가 중간에서 전환함

판단: ✅ 흐름이 연결됨
  → "源浚流淸"의 구조
  → 木(근원) → 火(흐름) → 土(결과)

```

#### 6-3-2. 병약론 적용

##### 조건 4: 병약(病藥) 이론 적용

```
【적천수 병약론의 핵심】

적천수의 핵심 중 하나:
  "병을 찾고 약을 쓰는 것"
  
  → 사주에 병(病)이 있으면
  → 그 병을 치료하는 약(藥)이 용신
  → 병약론이 용신 선정의 핵심

【이 사주의 병(病) 찾기】

질문: 이 사주의 병(病)은 무엇인가?

Step 1~5 종합:
  - 월령: 寅木이 戊土를 극함 (木克土)
  - 시간: 甲木이 투출하여 일간을 극함
  - 甲木의 세력:
    * 시간 투출 ✅
    * 寅寅 본기 통근 (이중 통근) ✅
    * 월령 득세 ✅
    → 매우 강함

병(病)의 정체:
  - 甲木의 과다한 克 (편관 과다)
  - 일간 戊土를 직접 극하는 기운
  - 월령과 시간에서 모두 木氣가 강함
  → 편관(偏官)이 과다하여 일간을 압박함

【이 사주의 약(藥) 찾기】

질문: 이 병을 치료하는 약(藥)은 무엇인가?

약(藥)의 조건:
  - 병(甲木의 과다한 克)을 치료할 수 있어야 함
  - 일간을 돕고 병을 제어해야 함

약(藥)의 작용:
  - 丙火가 甲木을 설기(泄氣)함 (木生火)
  - 丙火가 戊土를 생조함 (火生土)
  - 甲木 → 丙火 → 戊土 경로로 전환
  → 克을 生으로 전환함

약(藥)의 정체:
  - 丙火 (월간 투출 + 강한 통근)
  → 통관(通關)하여 克을 生으로 전환
  → 병(甲木의 克)을 치료하는 약

【병약론 종합】

병(病):
  - 甲木의 과다한 克 (편관 과다)
  - 일간 戊土를 직접 극하는 기운
  - 월령과 시간에서 모두 강함

약(藥):
  - 丙火 (통관하여 克을 生으로 전환)
  - 甲木의 기운을 받아 戊土를 생함
  - 병을 치료하여 중화를 이룸

병약 관계:
  - 병(甲木)이 있으나
  - 약(丙火)이 있어 치료됨 ✅
  → 병약론에 의하면 용신은 丙火

```

```

### 7-5. 격국(格局) 논의

```
【격국(格局)의 개념】

적천수는 격국론도 중시함
→ 월령을 기준으로 격국을 정함
→ 격국에 따라 용신 활용법이 달라짐

【이 사주의 격국 판단】

월령 기준:
  - 월지: 寅 (木)
  - 월간: 丙火 (투출)
  - 일간: 戊土

격국 후보:

① 편관격(偏官格):
  - 조건: 월령이 편관(七殺)
  - 이 사주: 寅은 戊土의 편관
  - 그러나 丙火가 월간에 투출하여 편인격 성립
  → 편관격 아님

② 편인격(偏印格):
  - 조건: 월령이 편인(偏印)
  - 이 사주: 寅은 戊土의 편관이지만
  - 丙火가 월간에 투출하여 편인 작용
  - 다만 월령 자체는 편관이므로 완전한 편인격 아님

③ 편관생인격(偏官生印格):
  - 조건: 편관이 편인을 생함
  - 이 사주: 甲木(편관) → 丙火(편인) → 戊土(일간)
  - 편관이 편인을 생하고, 편인이 일간을 생함
  → 이 격국에 해당 ✅

【격국 종합 판단】

격국: 편관생인격(偏官生印格)
  - 편관(甲木)이 편인(丙火)을 생함
  - 편인(丙火)이 일간(戊土)를 생함
  - 상생의 구조로 길함

격국 특성:
  - 편관이 강하나 편인으로 전환됨
  - 편인이 일간을 생조하여 길함
  - 용신은 편인(丙火)

【격국과 용신의 관계】

편관생인격에서:
  - 용신: 편인(丙火) ✅
  - 편인을 돕는 것: 火 (丁火)
  - 편인을 생하는 것: 木 (甲木, 乙木) - 단, 과하지 않게
  - 편인을 극하는 것: 水 (壬水, 癸水) - 기신

이 사주:
  - 용신: 丙火 (편인) ✅
  - 격국에 맞는 용신 선정

```

### 7-6. 희용신(喜用神) 순서 정리

```
【희용신의 개념】

用神: 가장 긴요한 것 (용신)
喜神: 용신을 돕거나 용신이 생하는 것
忌神: 용신을 극하거나 용신이 극하는 것

【이 사주의 희용신 정리】

【用神】 火 (丙火, 丁火)
  → 가장 긴요한 것
  → 4가지 용신 유형 모두 수렴
  → 조후, 부억, 통관, 병약 모두 담당

【喜神】 火 (丁火)
  → 용신(丙火)을 돕는 것
  → 같은 火로 힘을 보탬
  → 용신과 같은 오행

【喜神】 木 (甲木, 乙木) - 조건부
  → 용신(丙火)을 생함 (木生火)
  → 단, 과하지 않게
  → 현재는 甲木이 강하나 丙火로 전환되어 유리함
  → 다만 木이 과도하면 용신을 설기할 수 있음

【喜神】 土 (戊土, 己土)
  → 용신(火)이 생하는 대상
  → 일간과 같은 편
  → 힘을 보탬

【忌神】 水 (壬水, 癸水)
  → 용신(火)를 극함 (水克火)
  → 용신이 상하면 중화가 깨짐
  → 가장 경계해야 할 것

【忌神】 木 (甲木, 乙木) - 과다 시
  → 용신(火)을 과도하게 설기함
  → 현재는 적절하나 과도하면 흉함

【閑神】 金 (庚金, 辛金)
  → 용신과 직접 관계 없음
  → 있어도 없어도 큰 영향 없음

【희용신 우선순위】

1순위: 用神 火 (丙火, 丁火) - 최우선
2순위: 喜神 火 (丁火) - 용신을 돕는 것
3순위: 喜神 土 (戊土, 己土) - 용신이 생하는 대상
4순위: 喜神 木 (甲木, 乙木) - 조건부 (과하지 않게)
5순위: 忌神 水 (壬水, 癸水) - 가장 경계
6순위: 忌神 木 (과다 시) - 경계
7순위: 閑神 金 (庚金, 辛金) - 영향 적음

```

### 7-7. 대운/세운 적용 지침

```
【대운/세운의 개념】

대운(大運): 10년 단위 운세
세운(歲運): 1년 단위 운세
→ 용신이 대운/세운에서 어떻게 작용하는지가 중요

【용신(火) 기준 대운/세운 판단】

① 火運 (丙火, 丁火) = 길(吉)
  - 용신이 강해짐
  - 조후, 부억, 통관, 병약 모두 강화됨
  - 중화가 더욱 완성됨
  → 가장 길한 운

② 土運 (戊土, 己土) = 길(吉)
  - 용신이 생하는 대상
  - 일간이 강해짐
  - 용신의 작용이 더욱 효과적
  → 길한 운

③ 木運 (甲木, 乙木) = 반길반흉(半吉半凶)
  - 용신을 생함 (木生火) → 길함
  - 다만 과도하면 용신을 설기할 수 있음 → 흉함
  - 현재 사주에 木이 이미 강함
  → 적절하면 길, 과도하면 흉

④ 金運 (庚金, 辛金) = 평(平)
  - 용신과 직접 관계 없음
  - 큰 변화 없음
  → 평범한 운

⑤ 水運 (壬水, 癸水) = 흉(凶)
  - 용신(火)를 극함 (水克火)
  - 용신이 상하면 중화가 깨짐
  - 병(甲木의 克)이 다시 나타날 수 있음
  → 가장 흉한 운

【대운/세운 적용 예시】

예시 1: 火運 (丙火, 丁火)
  → 용신 강화 → 길(吉)
  → 사업, 인연, 건강 모두 좋음

예시 2: 水運 (壬水, 癸水)
  → 용신 상함 → 흉(凶)
  → 사업 실패, 건강 악화, 인연 불리

예시 3: 木運 (甲木, 乙木)
  → 용신 생함 → 길함
  → 다만 과도하면 설기 → 흉함
  → 적절한 수준 유지 필요

【대운/세운 판단 원칙】

1. 용신(火)을 강화하는 운 = 길
2. 용신(火)을 돕는 운 = 길
3. 용신(火)을 극하는 운 = 흉
4. 용신(火)을 설기하는 운 = 반길반흉 (과도 시 흉)

```

### 7-7-1. 대운/세운의 합충형(合沖刑) 영향

```
【원국 지지와 대운의 관계】

원국 지지: 巳(년) 寅(월) 戌(일) 寅(시)

【주요 합충형 대운】

① 午운 (매우 길):
   → 寅午戌 삼합(三合) 火局 완성
   → 용신(火) 극도로 강화
   → 가장 길한 대운

② 戌운 (길):
   → 寅戌 반합 강화
   → 火氣 모임
   → 길한 대운

③ 申운 (흉):
   → 寅申沖 (월지/시지 충돌)
   → 용신의 통근지(寅) 손상
   → 기반 흔들림

④ 亥운 (매우 흉):
   → 巳亥沖 (년지 충돌)
   → 용신의 본기 통근지(巳) 손상
   → 亥 = 壬水 본기 → 용신 직접 극
   → 가장 흉한 대운

⑤ 卯운 (반길반흉):
   → 卯戌合 (일지와 합)
   → 木氣 증가
   → 용신 생하나 과다 주의

【합충형 판단 원칙】

1. 용신(火)을 강화하는 합충형 = 길
   → 午운 (삼합 완성), 戌운 (반합 강화)

2. 용신의 통근지를 손상하는 충형 = 흉
   → 申운 (寅 충돌), 亥운 (巳 충돌)

3. 기신을 강화하는 합충형 = 경계
   → 卯운 (木氣 증가)
