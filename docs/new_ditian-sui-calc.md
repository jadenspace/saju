# 용신 7단계 프로세스

---

## Step 1: 월령 심사 (月令 審査)

> 핵심 질문: "월령이 일간에게 어떤 존재인가?"
> 

---

### 1-1. 적천수 원문

**원문 1:**

> "月令乃提綱之府, 譬之宅也, 人元為用事之神"
(월령은 제강의 부이니 집에 비유하고, 인원은 용사의 신이다)
> 

**원문 2:**

> "五陽從氣不從勢, 五陰從勢無情義"
(오양은 기를 따르고 세를 따르지 않으며, 오음은 세를 따르되 정의가 없다)
> 

---

### 1-2. 원문 해석

### "月令乃提綱之府"

```
提綱 = 벼리(그물의 큰 줄), 강령
府 = 관청, 근본이 되는 곳

해석:
  월령은 사주 전체를 통솔하는 강령의 자리
  모든 판단의 출발점

```

### "譬之宅也"

```
譬 = 비유하다
宅 = 집

해석:
  월령을 집에 비유함
  → 일간이 어떤 환경(집)에 거주하는가
  → 그 집이 나에게 편한가, 불편한가

```

### "人元為用事之神"

```
人元 = 지장간 (지지 속에 숨은 천간)
用事 = 일을 주관함
神 = 주체

해석:
  지장간 중 사령하는 글자가 실제 주관자
  → 월지의 겉모습이 아니라 지장간을 봐야 함

```

---

### 1-3. 이 사주에 적용

### 월령 확인

| 항목 | 내용 |
| --- | --- |
| 일간 | 戊土 |
| 월지 | 寅木 |

### "月令乃提綱之府" 적용

```
질문: 월령 寅이 일간 戊에게 어떤 집인가?

寅 = 木의 자리
戊 = 土

木과 土의 관계: 木克土 (목이 토를 극함)

해석:
  戊土가 寅木의 집에 거주함
  → 나를 극하는 환경에 있음
  → 편하지 않은 집

```

### "人元為用事之神" 적용

```
질문: 寅의 지장간 중 누가 용사(用事)하는가?

寅의 지장간: 戊 → 丙 → 甲

본기(本氣): 甲木
→ 甲木이 寅월을 주관하는 신(神)

甲木 = 戊土의 편관 (나를 극하는 존재)

해석:
  월령을 주관하는 자가 나를 극하는 편관
  → 월령의 주인이 나에게 불리함

```

### "五陽從氣不從勢" 적용

```
戊土 = 오양(五陽) 중 하나 (甲丙戊庚壬)

적천수 원문:
  "오양은 기(氣)를 따르고 세(勢)를 따르지 않는다"

해석:
  戊土는 양간이므로:
  → 겉으로 보이는 세력(數)보다
  → 월령의 기운(氣)을 중시해야 함

  월령 寅의 기운: 木氣 (나를 극하는 기운)
  → 양간 戊土는 이 木氣의 영향을 강하게 받음

```

---

### 1-4. 월지 지장간 투출 확인

**적천수 관점:**

```
"人元為用事之神"
→ 지장간이 천간에 투출하면 그 작용이 드러남

```

**寅의 지장간 투출 여부:**

| 지장간 | 천간 투출 | 비고 |
| --- | --- | --- |
| 戊土 | ❌ | 己는 있으나 戊는 없음 |
| 丙火 | ✅ **월간** | 투출하여 작용 드러남 |
| 甲木 | ✅ **시간** | 투출하여 작용 드러남 |

```
판단:
  - 丙火(편인): 월간 투출 → 나를 생하는 작용이 드러남
  - 甲木(편관): 시간 투출 → 나를 극하는 작용이 드러남

  월령 속에 나를 돕는 것(丙)과 해치는 것(甲)이 공존
  둘 다 천간에 드러나 작용함

```

## Step 2: 조후 심사 (調候 審査)

> 핵심 질문: "천도의 한난(寒暖)이 일간에게 어떤 영향을 주는가?"
> 

---

### 2-1. 적천수 원문

**원문:**

> "天道有寒暖, 發育萬物, 人道得之, 不可過也"
(천도에 한난이 있어 만물을 발육하니, 인도가 이를 얻되 지나침이 없어야 한다)
> 

---

### 2-2. 원문 해석

### "天道有寒暖"

```
天道 = 하늘의 이치, 자연의 법칙
寒暖 = 차가움과 따뜻함

해석:
  자연에는 춥고 따뜻한 기후가 있음
  → 계절에 따른 기후 변화
  → 월령(계절)이 가진 기후적 성질

```

### "發育萬物"

```
發育 = 발생하고 기름
萬物 = 만물

해석:
  한난의 조화가 만물을 자라게 함
  → 적절한 한난이 있어야 생명이 유지됨
  → 너무 춥거나 더우면 생육이 어려움

```

### "人道得之, 不可過也"

```
人道 = 사람의 도리, 사주에서는 일간
得之 = 이를 얻음
不可過 = 지나침이 없어야 함

해석:
  일간이 한난의 조화를 얻어야 함
  → 지나치게 춥거나 덥지 않아야 함
  → 극단적 기후는 일간에게 해로움

```

---

### 2-3. 이 사주에 적용

### 조건 1: 월령의 한난(寒暖) 확인

```
월지: 寅 (정월, 입춘 후)

질문: 寅월의 천도는 어떠한가?

寅월의 기후:
  - 입춘 직후
  - 겨울이 끝나고 봄이 시작됨
  - 아직 한기(寒氣)가 남아있음

판단: 寒 (차가운 쪽)
  → "天道有寒暖" 중 寒에 해당

```

### 조건 2: 일간이 한난을 어떻게 받는가

```
일간: 戊土

질문: 戊土가 寒한 환경에서 어떠한가?

土의 성질:
  - 차가우면 굳고 얼어붙음
  - 따뜻해야 부드럽고 만물을 기름

戊土 + 寒한 寅월:
  → 흙이 아직 차가운 상태
  → 따뜻함(暖)이 필요함

판단: 暖을 얻어야 "發育萬物"할 수 있음

```

### 조건 3: "不可過也" - 지나침 여부 확인

```
질문: 이 사주에서 한난이 지나친가?

寅월의 寒:
  - 극한(極寒)인 子월, 丑월이 아님
  - 초춘으로 한기가 있으나 극단적이지 않음

판단: 過하지 않음
  → 치명적 수준은 아님
  → 그러나 暖의 도움이 있으면 좋음

```

### 조건 4: 暖을 주는 것이 사주에 있는가

```
질문: 사주 내에 따뜻함(暖)을 주는 것이 있는가?

暖을 주는 오행: 火

사주 내 火:
  - 천간: 丙火 (월간) ✅
  - 지지: 巳火 (년지) ✅

丙火의 상태:
  - 월간에 투출 (드러나 작용함)
  - 巳에 본기 통근 (뿌리 있음)
  - 寅에 중기 통근 (추가 뿌리)

판단: ✅ 暖이 갖춰짐
  → "人道得之" 충족

```

## Step 3: 쇠왕 심사 (衰旺 審査)

> 핵심 질문: "일간의 쇠왕의 진기(眞機)는 어떠한가?"
> 

---

### 3-1. 적천수 원문

**원문:**

> "能知衰旺之真機, 其於三命之奧, 思過半矣"
(쇠왕의 진기를 알면 삼명의 오묘함을 절반 이상 안 것이다)
> 

---

### 3-2. 원문 해석

### "衰旺"

```
衰 = 쇠함, 약함
旺 = 왕성함, 강함

해석:
  일간의 힘이 쇠한가, 왕한가
  → 단순한 강약이 아닌 실질적 상태

```

### "真機"

```
真 = 참된, 진짜의
機 = 기틀, 핵심 작용

해석:
  겉으로 보이는 것이 아닌 참된 기틀
  → 오행 개수가 아닌 실제 작용하는 힘
  → 표면이 아닌 본질

```

### "三命之奧, 思過半矣"

```
三命 = 명리학 전체
奧 = 오묘함, 깊은 이치
思過半 = 절반 이상을 생각함

해석:
  쇠왕의 진기를 알면 명리의 절반 이상을 아는 것
  → 그만큼 쇠왕 판단이 핵심이라는 뜻
  → 그러나 "진기"를 알아야 함 (겉모습 아님)

```

---

### 3-3. 이 사주에 적용

### 조건 1: 월령에서의 쇠왕

```
질문: 일간이 월령에서 왕한가, 쇠한가?

일간: 戊土
월령: 寅木 (木이 사령)

월령과의 관계:
  - 寅월은 木이 왕성한 때
  - 木은 土를 극함 (木克土)
  - 일간이 월령에서 극을 받음

판단: 衰 (월령에서 쇠함)

```

### 조건 2: 진기(眞機) 파악 - 겉과 속

```
질문: 겉으로 쇠해 보이나, 진기는 어떠한가?

"眞機"를 보려면:
  - 겉모습(월령 실령)만 보지 않음
  - 실제로 일간이 작용할 힘이 있는가를 봄

확인할 것:
  ① 일간이 뿌리(근)가 있는가?
  ② 일간을 돕는 것이 있는가?
  ③ 일간을 해치는 것이 제어되는가?

```

### 조건 3: 일간의 뿌리(根) 확인

```
질문: 戊土가 지지에 뿌리가 있는가?

지지 검토:
  - 년지 巳: 지장간 戊·庚·丙 → 戊 있음 ✅
  - 월지 寅: 지장간 戊·丙·甲 → 戊 있음 ✅
  - 일지 戌: 지장간 辛·丁·戊 → 戊 있음 (본기) ✅
  - 시지 寅: 지장간 戊·丙·甲 → 戊 있음 ✅

특히 일지(坐下):
  - 戌土에 戊土 본기
  - 일간이 앉은 자리에 강한 뿌리

판단: 根이 있음
  → 겉으로 쇠하나 뿌리가 있어 무너지지 않음

```

### 조건 4: 일간을 돕는 것 확인

```
질문: 천간에서 일간을 돕는 것이 있는가?

천간 검토:
  - 년간 己土: 같은 土로 힘을 보탬 ✅
  - 월간 丙火: 火生土로 일간을 생함 ✅
  - 시간 甲木: 木克土로 일간을 극함 ❌

돕는 것: 己土, 丙火
해치는 것: 甲木

판단: 돕는 것이 있음
  → 己土가 힘을 보태고
  → 丙火가 생조함

```

### 조건 5: 해치는 것의 상태 확인

```
질문: 일간을 해치는 甲木은 어떤 상태인가?

甲木의 힘:
  - 시간에 투출
  - 寅寅에 본기 통근 (강한 뿌리)
  - 월령을 얻어 왕성함

甲木의 작용:
  - 직접 戊土를 극하는가?
  - 아니면 다른 경로가 있는가?

확인:
  - 甲木 → 丙火 → 戊土 경로 존재
  - 甲이 丙을 생하고 (木生火)
  - 丙이 戊를 생함 (火生土)
  - 중간에 丙火가 있어 전환됨

판단: 해치는 것이 전환됨
  → 甲木이 강하나 丙火가 받아줌
  → 직접적 극이 완화됨

```

---

### 3-4. 진기(眞機) 종합 판정

```
적천수 "眞機" 기준:

겉모습:
  - 월령 실령 → 쇠해 보임

진기(眞機):
  - 일지에 본기 통근 → 뿌리 있음
  - 己土, 丙火의 도움 → 생조 받음
  - 甲木의 극이 丙火로 전환 → 해침이 완화됨

결론:
  "衰하나 眞機가 있음"
  → 겉으로 쇠하나 실제로 무너지지 않는 상태

```

## Step 4: 종격 심사 (從格 審査)

> 핵심 질문: "일간이 종(從)할 것인가, 종하지 않을 것인가?"
> 

---

### 4-1. 적천수 원문

**원문 1:**

> "從得真者只論從, 從神又有吉和凶"
(종이 참됨을 얻으면 오직 종으로 논하고, 종신에도 길흉이 있다)
> 

**원문 2:**

> "假從亦可例言從, 見從不從反為凶"
(가종도 종으로 논할 수 있으나, 종인 듯 종이 아니면 도리어 흉하다)
> 

---

### 4-2. 원문 해석

### "從得真者只論從"

```
從 = 따름, 복종함
得真 = 참됨을 얻음
只論從 = 오직 종으로 논함

해석:
  종격이 참되게 성립하면
  → 다른 것 볼 필요 없이 종으로만 판단
  → 종신(따르는 대상)이 용신이 됨

```

### "從神又有吉和凶"

```
從神 = 종하는 대상 (따르는 신)
吉和凶 = 길함과 흉함

해석:
  종격이 성립해도 종신의 상태에 따라 길흉이 갈림
  → 종신이 온전하면 길
  → 종신이 파손되면 흉

```

### "假從亦可例言從"

```
假從 = 가짜 종격
亦可 = 또한 가능하다
例言從 = 종으로 논함

해석:
  완전한 종격이 아니어도 종으로 볼 수 있는 경우
  → 거의 종격에 가까운 상태

```

### "見從不從反為凶"

```
見從不從 = 종인 듯 종이 아님
反為凶 = 도리어 흉함

해석:
  종격처럼 보이나 실제로 종이 아닌 경우
  → 억지로 종으로 보면 흉함
  → 진종(眞從)과 가종(假從)을 구별해야 함

```

---

### 4-3. 이 사주에 적용

### 조건 1: 종(從)할 상황인가?

```
질문: 일간이 종해야 할 만큼 무력한가?

종격의 전제:
  - 일간이 극히 약하여 스스로 버틸 수 없음
  - 한 세력이 압도적으로 강함
  - 일간을 돕는 것이 없거나 무력함

Step 3 결과 확인:
  - 일간 戊土: 衰하나 眞機 있음
  - 일지 戌에 본기 뿌리 있음
  - 丙火, 己土의 생조 있음

판단: ❌ 종할 상황 아님
  → 일간이 무력하지 않음
  → 뿌리와 생조가 있어 버틸 수 있음

```

### 조건 2: "從得真者" 체크

```
질문: 종격이 참되게 성립하는가?

진종(眞從)의 조건:
  ① 일간이 뿌리가 전무하거나 극히 미약
  ② 일간을 돕는 인비가 없거나 무력
  ③ 한 오행이 압도적으로 강함

체크:
  ① 뿌리: 戌土 본기 + 巳寅寅에 여기 → ❌ 뿌리 있음
  ② 인비: 丙火 투출 + 통근, 己土 투출 → ❌ 유력한 인비 있음
  ③ 압도: 木이 강하나 火土도 버팀 → ❌ 완전 압도 아님

판단: ❌ 진종 불성립
  → 세 조건 모두 충족 안 됨

```

### 조건 3: "假從亦可例言從" 체크

```
질문: 가종(假從)으로라도 볼 수 있는가?

가종의 조건:
  - 진종은 아니나 거의 종에 가까움
  - 일간의 뿌리가 아주 미약함
  - 인비가 있으나 힘이 약함

이 사주:
  - 일지 戌에 본기 통근 → 미약하지 않음
  - 丙火가 월간 투출 + 강한 통근 → 유력함

판단: ❌ 가종도 불성립
  → 일간의 뿌리와 인비가 유력함

```

### 조건 4: "見從不從反為凶" 확인

```
질문: 이 사주를 종격으로 보면 어떻게 되는가?

만약 종격으로 본다면:
  - 木을 종신으로 삼아야 함
  - 火土를 기신으로 봐야 함

그러나 실제:
  - 戊土가 戌에 뿌리 있음
  - 丙火가 유력하게 생조함
  - 己土가 힘을 보탬

"見從不從反為凶":
  → 종이 아닌데 종으로 보면 흉함
  → 이 사주는 종으로 보면 안 됨

판단: 종격 아님을 확정

```

## Step 5: 희기 심사 (喜忌 審査)

> 핵심 질문: "무엇이 이 사주에 이롭고, 무엇이 해로운가?"
> 

---

### 5-1. 적천수 원문

**원문 1:**

> "何知其人吉, 喜神爲輔持"
(그 사람이 길한 줄 어찌 아는가, 희신이 보좌하기 때문이다)
> 

**원문 2:**

> "何知其人凶, 忌神輾轉攻"
(그 사람이 흉한 줄 어찌 아는가, 기신이 번갈아 공격하기 때문이다)
> 

**원문 3:**

> "何知其人壽, 性定元氣厚"
(그 사람이 장수하는 줄 어찌 아는가, 성품이 안정되고 원기가 두텁기 때문이다)
> 

---

### 5-2. 원문 해석

### "何知其人吉, 喜神爲輔持"

```
何知 = 어찌 아는가
吉 = 길함
喜神 = 기뻐하는 신, 이로운 것
輔持 = 보좌하여 지탱함

해석:
  사람이 길한 것은 희신이 보좌하기 때문
  → 희신이 있고, 그것이 일간을 돕고 있어야 함
  → 희신이 있어도 작용하지 못하면 소용없음

```

### "何知其人凶, 忌神輾轉攻"

```
凶 = 흉함
忌神 = 꺼리는 신, 해로운 것
輾轉 = 이리저리 구름, 번갈아
攻 = 공격함

해석:
  사람이 흉한 것은 기신이 번갈아 공격하기 때문
  → 기신이 제어 없이 일간을 해침
  → 기신이 있어도 제어되면 흉하지 않음

```

### "性定元氣厚"

```
性定 = 성품이 안정됨
元氣厚 = 원기가 두터움

해석:
  사주의 기운이 안정되고 두터우면 좋음
  → 한쪽으로 치우치지 않음
  → 희신이 보좌하고 기신이 제어된 상태

```

---

### 5-3. 이 사주에 적용

### 조건 1: 희신(喜神) 후보 파악

```
질문: 이 사주에서 일간을 이롭게 하는 것은 무엇인가?

일간 戊土에게 이로운 작용:
  ① 같은 오행 → 힘을 보탬
  ② 나를 생하는 오행 → 생조함

사주 내 확인:
  - 丙火 (월간): 火生土 → 일간을 생함 ✅
  - 己土 (년간): 같은 土 → 힘을 보탬 ✅

희신 후보: 丙火, 己土

```

### 조건 2: 기신(忌神) 후보 파악

```
질문: 이 사주에서 일간을 해롭게 하는 것은 무엇인가?

일간 戊土에게 해로운 작용:
  ① 나를 극하는 오행 → 압박함
  ② 나를 설기하는 오행 → 기운을 뺌

사주 내 확인:
  - 甲木 (시간): 木克土 → 일간을 극함 ⚠️
  - 寅木 (월지, 시지): 甲의 뿌리 → 기신의 근원 ⚠️

기신 후보: 甲木, 寅木

```

### 조건 3: "喜神爲輔持" 체크

```
질문: 희신이 실제로 보좌(輔持)하고 있는가?

輔持의 조건:
  ① 천간에 드러나 작용함
  ② 지지에 뿌리가 있어 안정됨
  ③ 극파되지 않아 온전함
  ④ 일간에게 실질적 도움을 줌

丙火 체크:
  ① 투출: 월간에 투출 ✅
  ② 통근: 巳(본기), 寅(중기) ✅
  ③ 극파: 壬癸水 없음 ✅
  ④ 작용: 火生土로 일간 생조 ✅

판단: ✅ 丙火가 輔持하고 있음

己土 체크:
  ① 투출: 년간에 투출 ✅
  ② 통근: 戌(여기) ✅
  ③ 극파: 직접 극하는 것 없음 ✅
  ④ 작용: 같은 土로 힘을 보탬 ✅

판단: ✅ 己土도 輔持하고 있음 (보조)

```

### 조건 4: "忌神輾轉攻" 체크

```
질문: 기신이 번갈아 공격(輾轉攻)하고 있는가?

輾轉攻의 조건:
  - 기신이 제어 없이 일간을 계속 해침
  - 막아주는 것 없이 직접 공격함

甲木 체크:

甲木의 힘:
  - 시간에 투출 ✅
  - 寅寅에 본기 통근 (강함) ✅
  - 월령을 얻음 ✅
  → 기신의 힘은 강함

甲木의 작용 경로:
  - 경로 A: 甲 → 戊 (직접 극)
  - 경로 B: 甲 → 丙 → 戊 (전환)

현재 상태:
  - 丙火가 월간에 있음 (甲과 戊 사이)
  - 甲木이 丙火를 생함 (木生火)
  - 丙火가 戊土를 생함 (火生土)
  → 경로 B가 작동함
  → 甲의 기운이 丙을 통해 전환됨

판단: ❌ 輾轉攻 아님
  → 기신(甲木)이 강하나
  → 희신(丙火)이 중간에서 받아 전환함
  → 직접 공격이 아닌 간접 생조로 바뀜

```

### 조건 5: 희기(喜忌) 확정

```
적천수 기준 최종 판단:

희신(喜神):
  - 丙火: "輔持"함 ✅
    → 월간 투출, 강한 통근
    → 일간을 생조하고 기신을 전환함
    → 핵심 희신

  - 己土: "輔持"함 ✅
    → 년간 투출, 통근 있음
    → 일간과 같은 편으로 힘을 보탬
    → 보조 희신

기신(忌神):
  - 甲木: 강하나 제어됨
    → "輾轉攻" 하지 못함
    → 丙火가 전환하여 해침이 완화됨

  - 寅木: 甲의 뿌리
    → 기신의 근원이나
    → 丙火도 寅에서 중기 통근하여 공유함

```

### 6-2. 원문 해석

### "中和之氣, 其福獨鍾"

```
中和 = 치우치지 않고 고르게 조화됨
之氣 = 의 기운
其福 = 그 복
獨鍾 = 홀로 모임

해석:
  중화의 기운을 이루면 복이 모임
  → 한쪽으로 치우치지 않음
  → 기운의 흐름이 막힘 없이 통함
  → 희신이 작용하고 기신이 제어된 상태

```

### "旺相休囚, 辨其虛實"

```
旺相休囚 = 왕, 상, 휴, 수 (오행의 네 가지 상태)
辨 = 분별함
虛實 = 허함과 실함

해석:
  오행의 상태를 분별하여 허실을 봄
  → 겉으로 왕해 보여도 속이 허할 수 있음
  → 겉으로 쇠해 보여도 속이 실할 수 있음
  → 진기(眞機)를 봐야 함

```

### "源浚流淸, 斯眞貴矣"

```
源浚 = 근원이 깊음
流淸 = 흐름이 맑음
斯 = 이것이
眞貴 = 참으로 귀함

해석:
  근원이 깊고 흐름이 맑으면 참으로 귀함
  → 오행의 흐름이 막힘 없이 연결됨
  → 생극의 관계가 순조로움

```

### 6-3. 이 사주에 적용

### 조건 1: "中和之氣" 체크 - 치우침 여부

```
질문: 이 사주가 한쪽으로 치우쳐 있는가?

오행 분포 확인:
  - 木: 甲(천간) + 寅寅(지지) → 강함
  - 火: 丙(천간) + 巳(지지) → 있음
  - 土: 戊己(천간) + 戌(지지) → 있음
  - 金: 지장간에만 → 약함
  - 水: 없음 → 결여

치우침 여부:
  - 木이 가장 강함 (월령 + 다수)
  - 그러나 火와 土도 버티고 있음
  - 완전한 한쪽 치우침은 아님

판단: △ 木이 강하나 극단적 치우침은 아님

```

### 조건 2: "旺相休囚, 辨其虛實" 체크

```
질문: 각 오행의 허실(虛實)은 어떠한가?

木의 허실:
  - 겉: 월령 득세, 寅寅 통근 → 旺해 보임
  - 속: 丙火가 木氣를 빼감 → 완전 實은 아님

火의 허실:
  - 겉: 천간 1개, 지지 1개 → 적어 보임
  - 속: 월간 투출 + 巳寅 통근 + 木의 생조 → 實함

土의 허실:
  - 겉: 월령 실령 → 衰해 보임
  - 속: 일지 본기 통근 + 火의 생조 → 虛하지 않음

판단:
  - 木: 旺하나 火에 설기되어 완전 實 아님
  - 火: 적어 보이나 實함
  - 土: 쇠해 보이나 虛하지 않음 (眞機 있음)

```

### 조건 3: "源浚流淸" 체크 - 흐름의 연결

```
질문: 오행의 흐름이 막힘 없이 연결되는가?

흐름 확인:

木 → 火 → 土 경로:
  甲木(시간) → 丙火(월간) → 戊土(일간)
  寅木(월지) → 巳火(년지) → 戌土(일지)

  - 木이 火를 생함 (木生火) ✅
  - 火가 土를 생함 (火生土) ✅
  - 상생으로 연결됨 ✅

막힘 여부:
  - 木과 土 사이에 火가 있음
  - 木이 土를 직접 극하지 않음
  - 火가 중간에서 전환함

판단: ✅ 흐름이 연결됨
  → "源浚流淸"의 구조
  → 木(근원) → 火(흐름) → 土(결과)

```

### 조건 4: 중화(中和) 성립 여부

```
질문: 이 사주가 중화의 기운을 이루는가?

중화의 조건:
  ① 극단적 치우침이 없음
  ② 허실이 분별되어 眞機가 있음
  ③ 흐름이 막힘 없이 연결됨
  ④ 희신이 보좌하고 기신이 제어됨

체크:
  ① 치우침: △ 木 강하나 극단 아님 ✅
  ② 허실: 土가 虛해 보이나 眞機 있음 ✅
  ③ 흐름: 木→火→土 상생 연결 ✅
  ④ 희기: 丙火 보좌, 甲木 제어됨 ✅

판단: ✅ 중화를 이룸

```

### 6-4. 중화의 핵심 - 丙火의 역할

```
질문: 중화를 이루는 핵심은 무엇인가?

丙火의 역할:

① 천도(天道) 측면:
   → 寅월의 寒을 녹여 暖을 줌
   → "天道有寒暖, 發育萬物" 충족

② 희기(喜忌) 측면:
   → 일간을 생조하여 보좌 (輔持)
   → 기신(甲木)을 전환하여 공격 방지

③ 중화(中和) 측면:
   → 木과 土 사이에서 흐름을 연결
   → "源浚流淸"의 핵심 통로

결론:
  丙火가 없으면 이 사주는 중화를 이루지 못함
  → 木이 土를 직접 극하여 치우침
  → 寒이 해소되지 않아 발육 못함
  → 기신이 輾轉攻하여 흉함

  丙火가 있어 중화를 이룸
  → "中和之氣, 其福獨鍾"

```

### 7-2. 원문 해석

### "喜神爲輔持"와 용신의 관계

```
적천수에서 "용신"이라는 단어:
  → 직접적으로 "用神" 용어를 정의하지 않음
  → 대신 "喜神"의 개념을 사용
  → 喜神이 輔持하는 것이 곧 긴요한 것

용신의 적천수적 정의:
  → 희신 중에서 가장 핵심적인 역할을 하는 것
  → 없으면 사주가 중화를 이루지 못하는 것
  → 사주의 병(病)을 치료하는 약(藥)

```

### "性定元氣厚"

```
性定 = 성품이 안정됨
元氣厚 = 원기가 두터움

해석:
  용신이 제대로 작용하면:
  → 사주의 성품(구조)이 안정됨
  → 원기(근본 기운)가 두터워짐

```
