# 사주 명리학 신년운세 해석 체계

## 1. 운세 해석의 기본 구조

### 1-1. 세운(歲運) 해석의 3층 구조

```
┌─────────────────────────────────────────────────────┐
│                    세운(歲運) 분석                    │
├─────────────────────────────────────────────────────┤
│  1층: 세운 천간 → 일간과의 십신 관계 (정신/의지 영역)  │
│  2층: 세운 지지 → 사주 지지와의 형충회합 (현실 사건)   │
│  3층: 세운 납음오행 → 전체적 기운의 흐름              │
└─────────────────────────────────────────────────────┘
```

### 1-2. 해석의 우선순위

| 순위 | 요소 | 영향력 | 설명 |
|------|------|--------|------|
| 1 | **대운(大運)** | 40% | 10년 단위의 큰 흐름 |
| 2 | **세운(歲運)** | 30% | 해당 년도의 기운 |
| 3 | **월운(月運)** | 15% | 월별 세부 흐름 |
| 4 | **일운(日運)** | 10% | 일별 미세 변화 |
| 5 | **시운(時運)** | 5% | 시간대별 변화 |

## 2. 핵심 해석 도구

### 2-1. 십신(十神) 체계

십신은 **일간(나 자신)**을 기준으로 다른 글자와의 관계를 정의합니다.

```
일간 기준 십신 도출 원리:

┌──────────┬──────────────────────────────────────┐
│  관계    │              십신 명칭               │
├──────────┼──────────────────────────────────────┤
│ 나를 生  │ 정인(正印) / 편인(偏印)              │
│ 내가 生  │ 식신(食神) / 상관(傷官)              │
│ 나를 剋  │ 정관(正官) / 편관(偏官, 七殺)        │
│ 내가 剋  │ 정재(正財) / 편재(偏財)              │
│ 나와 同  │ 비견(比肩) / 겁재(劫財)              │
└──────────┴──────────────────────────────────────┘

※ 음양이 같으면 "편", 다르면 "정"
```

### 2-2. 십신별 상징 영역

| 십신 | 상징 | 직장 | 연애 | 금전 | 건강 |
|------|------|------|------|------|------|
| **정관** | 명예, 규율 | 승진, 안정 | 여성→배우자 | 고정수입 | 스트레스 |
| **편관** | 권력, 압박 | 경쟁, 도전 | 여성→외도 | 강제지출 | 사고, 수술 |
| **정인** | 학문, 모성 | 자격증, 학습 | 정신적 교류 | 부동산 | 휴식, 회복 |
| **편인** | 변화, 고독 | 이직, 전환 | 비밀연애 | 투기손실 | 신경계 |
| **비견** | 동료, 경쟁 | 협력, 독립 | 경쟁자 출현 | 동업 | 과로 |
| **겁재** | 다툼, 손실 | 갈등 | 삼각관계 | 사기, 손재 | 출혈 |
| **식신** | 표현, 재능 | 창작, 기술 | 낭만, 데이트 | 정당소득 | 소화기, 장수 |
| **상관** | 반항, 창의 | 이직충동 | 자유연애 | 변동수입 | 신경과민 |
| **정재** | 안정재물 | 월급, 보너스 | 남성→배우자 | 저축, 이자 | 과로누적 |
| **편재** | 투자, 횡재 | 사업확장 | 남성→외도 | 투자, 도박 | 간, 피로 |

## 3. 각 운세별 해석 기준

### 3-1. 총운(總運) 해석 기준

**핵심 질문**: "올해 전반적인 삶의 방향과 에너지는 어떠한가?"

#### 판단 요소

```
1. 세운 십신의 성격
   └─ 세운 천간이 일간에게 어떤 십신인가?
   
2. 용신(用神) 충족도
   └─ 세운의 오행이 현재 사주 구조에서 균형을 잡아주는 방향으로 작동하는가?
   
3. 기신(忌神) 발동 여부
   └─ 세운이 나에게 해로운 오행을 강화하는가?
   
4. 형충회합(刑沖會合) 작용
   └─ 세운 지지가 사주 지지와 어떤 관계인가?
   
5. 대운과의 조화
   └─ 현재 대운과 세운이 서로 돕는가, 충돌하는가?
```

#### 총운 판단 매트릭스

| 조건 | 긍정 신호 | 부정 신호 |
|------|-----------|-----------|
| 용신 | 세운이 용신과 같은 오행 | 세운이 용신을 극함 |
| 십신 | 정관, 정인, 정재, 식신 | 편관, 겁재, 상관 |
| 지지 | 삼합, 육합 형성 | 충(沖), 형(刑) 발생 |
| 대운 | 대운과 세운이 상생 | 대운과 세운이 충돌 |

### 3-2. 직장운(職場運) 해석 기준

**핵심 질문**: "올해 사회적 지위와 직업 영역에서의 변화는?"

#### 핵심 판단 요소

```
1. 관성(官星) 상태 분석
   ├─ 정관: 승진, 안정, 명예
   ├─ 편관: 경쟁, 압박, 변동
   └─ 관성 부재: 자영업/프리랜서 유리

2. 인성(印星) 보조 여부
   ├─ 인성 → 관성 생조: 상사의 도움, 후원
   └─ 인성 부재: 독자적 역량 발휘 필요

3. 식상(食傷) 발현
   ├─ 식신: 전문성 인정, 기술직 유리
   ├─ 상관: 창의력 발휘, 기존 질서 도전
   └─ 상관견관(傷官見官): 직장 내 갈등, 이직

4. 월주(月柱)와 세운 관계
   └─ 월주 = 사회궁(社會宮): 세운과의 충합 확인
```

#### 직장운 해석 공식

```
직장운 = f(관성강도, 인성보조, 식상발현, 월주작용)

세부 계산:
┌─────────────────────────────────────────────────┐
│ 관성(官星) 작용                                  │
│  - 세운이 정관: 승진/책임 증가 (+)              │
│  - 세운이 편관: 압박/경쟁 심화 (±)              │
│  - 관성이 충(沖): 직장 변동 가능성              │
├─────────────────────────────────────────────────┤
│ 인성(印星) 보조                                  │
│  - 세운이 인성: 학습/자격취득 유리 (+)          │
│  - 인성생관(印星生官): 실력으로 승진 (+)        │
├─────────────────────────────────────────────────┤
│ 식상(食傷) 발현                                  │
│  - 세운이 식신: 능력 인정 (+)                   │
│  - 세운이 상관: 이직 충동, 갈등 (±)             │
│  - 상관견관: 상사와 마찰 (-)                    │
└─────────────────────────────────────────────────┘
```

### 3-3. 연애운(戀愛運) 해석 기준

**핵심 질문**: "올해 이성관계와 감정 영역의 흐름은?"

#### 성별에 따른 배우자궁(配偶者宮) 십신

```
┌──────────┬─────────────┬─────────────┐
│  성별    │  배우자 십신 │   외도 십신  │
├──────────┼─────────────┼─────────────┤
│  남성    │  정재(正財)  │  편재(偏財)  │
│  여성    │  정관(正官)  │  편관(偏官)  │
└──────────┴─────────────┴─────────────┘
```

#### 연애운 핵심 판단 요소

```
1. 배우자 십신 작용
   ├─ 세운에서 배우자 십신 등장 → 인연 가능성 ↑
   ├─ 배우자 십신이 용신 → 관계가 비교적 안정적으로 흘러가기 쉬움
   └─ 배우자 십신이 기신 → 감정 소모가 커질 수 있어 조율이 필요함

2. 일지(日支) = 배우자궁
   ├─ 세운 지지와 일지가 합(合) → 결혼/연애 성사
   ├─ 세운 지지와 일지가 충(沖) → 이별/갈등
   └─ 세운 지지와 일지가 형(刑) → 다툼/상처

3. 도화살(桃花煞) 발동
   ├─ 子午卯酉 = 도화의 지지
   ├─ 세운이 도화살 발동 → 이성 만남 ↑
   └─ 도화살 + 충 = 외도/삼각관계 주의

4. 홍염살(紅艶煞) / 천희성(天喜星)
   └─ 연애/결혼 기회의 보조 지표
```

#### 연애운 해석 매트릭스

| 상황 | 미혼자 해석 | 기혼자 해석 |
|------|-------------|-------------|
| 정재/정관 등장 | 결혼감 인연 | 배우자 복 상승 |
| 편재/편관 등장 | 짧은 인연 | 외도 주의 |
| 일지와 합 | 연애/결혼 | 화목, 임신 가능 |
| 일지와 충 | 이별 가능성 | 갈등, 별거 가능 |
| 도화살 발동 | 만남 많음 | 유혹 주의 |
| 비겁 강화 | 경쟁자 출현 | 시댁/처가 갈등 |

### 3-4. 재물운(金錢運) 해석 기준

**핵심 질문**: "올해 재물의 흐름과 경제적 기회는?"

#### 재성(財星) 분석 원리

```
재성(財星) = 일간이 극(剋)하는 오행

┌─────────────────────────────────────────────────┐
│  일간 오행  │  재성 오행  │      의미           │
├─────────────────────────────────────────────────┤
│    木      │     土      │  목극토 = 재물      │
│    火      │     金      │  화극금 = 재물      │
│    土      │     水      │  토극수 = 재물      │
│    金      │     木      │  금극목 = 재물      │
│    水      │     火      │  수극화 = 재물      │
└─────────────────────────────────────────────────┘
```

#### 재물운 핵심 판단 요소

```
1. 재성(財星) 상태
   ├─ 정재(正財): 급여, 이자, 안정적 수입
   ├─ 편재(偏財): 투자, 사업, 횡재성 수입
   ├─ 재성 과다: 돈에 매이는 삶, 처(妻) 관련 고민
   └─ 재성 부족: 안정적 수입원 부족

2. 식상생재(食傷生財) 구조
   ├─ 식신 → 재성: 노력의 정당한 대가
   ├─ 상관 → 재성: 창의력/기술로 수입
   └─ 식상 부재: 재성만 있으면 불로소득 가능

3. 비겁(比劫)과 재성 관계
   ├─ 비겁 강 + 재성: 경쟁 속 재물 획득
   ├─ 비겁 극재(剋財): 동업 실패, 손재수
   └─ 겁재 발동: 도난, 사기 주의

4. 관성(官星)과 재성 관계
   ├─ 재성생관(財星生官): 재물이 명예로 연결
   └─ 재관쌍미(財官雙美): 부와 귀를 함께 얻음
```

#### 재물운 해석 공식

```
재물운 = f(재성강도, 식상생재, 비겁극재, 관성연결)

┌─────────────────────────────────────────────────┐
│ [가점 요소]                                      │
│  • 세운이 정재/편재                      +2.0    │
│  • 세운이 식신/상관 (생재 가능)          +1.5    │
│  • 재성이 용신                           +1.5    │
│  • 대운에서 재성 흐름                    +1.0    │
├─────────────────────────────────────────────────┤
│ [감점 요소]                                      │
│  • 세운이 비견/겁재 (극재)               -2.0    │
│  • 재성이 충(沖)                         -1.5    │
│  • 재성이 기신                           -1.0    │
│  • 인성과재(印星過財): 명예 vs 돈 갈등   -0.5    │
└─────────────────────────────────────────────────┘
재성이 용신 방향일 경우
수익 구조가 비교적 안정적으로 유지되며,
재성이 기신 방향일 경우
수익보다 관리·조절이 더 중요한 해로 해석합니다.
```

### 3-5. 건강운(健康運) 해석 기준

**핵심 질문**: "올해 신체적/정신적 건강 상태와 주의점은?"

#### 오행별 장부 배속

```
┌────────┬────────────┬────────────┬─────────────────┐
│ 오행   │   장(臟)   │   부(腑)   │   신체 부위      │
├────────┼────────────┼────────────┼─────────────────┤
│  木    │   간(肝)   │   담(膽)   │ 눈, 근육, 손발  │
│  火    │  심(心)    │ 소장(小腸) │ 혀, 혈관, 정신  │
│  土    │  비(脾)    │  위(胃)    │ 입술, 살, 소화  │
│  金    │  폐(肺)    │ 대장(大腸) │ 코, 피부, 호흡  │
│  水    │  신(腎)    │ 방광(膀胱) │ 귀, 뼈, 생식기  │
└────────┴────────────┴────────────┴─────────────────┘
```

#### 건강운 핵심 판단 요소

```
1. 오행 균형 분석
   ├─ 과다(過多) 오행: 해당 장부 기능 항진/염증
   ├─ 부족(不足) 오행: 해당 장부 기능 저하/허약
   └─ 극(剋)받는 오행: 손상/질병 가능성

2. 일간 강약(强弱) 판단
   ├─ 신강(身强): 활동적이나 무리하기 쉬움
   ├─ 신약(身弱): 면역력 저하, 만성피로
   └─ 태과/태약: 건강 문제 발생 확률 ↑

3. 충(沖)/형(刑) 작용
   ├─ 지지 충: 해당 오행 장부 손상, 사고
   ├─ 삼형(三刑): 수술, 입원 가능성
   └─ 자형(自刑): 자해, 정신건강 주의

4. 특정 십신 작용
   ├─ 식신(食神): 장수, 건강의 별 (식신이 상하면 건강 ↓)
   ├─ 편관(偏官): 사고, 수술, 외상
   ├─ 상관(傷官): 신경과민, 스트레스
   └─ 편인(偏印): 우울, 불면, 소화장애
```

#### 세운별 건강 주의사항 (2026 병오년 예시)

```
2026년 丙午年 = 火 기운 극강

┌─────────────────────────────────────────────────┐
│ [火 과다 시 주의]                                │
│  • 심장질환, 고혈압, 부정맥                      │
│  • 눈 충혈, 시력 저하                            │
│  • 조급함, 분노조절, 불면증                      │
│  • 피부 발진, 화상, 화재 사고                    │
├─────────────────────────────────────────────────┤
│ [火가 극하는 金(폐, 대장) 주의]                  │
│  • 호흡기 질환, 기관지염                         │
│  • 피부 트러블, 알레르기                         │
│  • 대장 문제, 변비/설사                          │
├─────────────────────────────────────────────────┤
│ [午 지지가 충하는 子(신장, 방광) 주의]           │
│  • 비뇨기 질환                                   │
│  • 허리, 뼈 관련 통증                            │
│  • 생식기 건강                                   │
└─────────────────────────────────────────────────┘
```

## 4. 통합 해석 프레임워크

### 4-1. 신년운세 해석 순서

```
Step 1: 원국(原局) 파악
        └─ 사주팔자의 기본 구조와
           해석의 기준점이 되는
           참고 용신/희신/기신 설정

Step 2: 대운(大運) 확인  
        └─ 현재 어떤 대운을 지나고 있는가?

Step 3: 세운(歲運) 분석
        ├─ 세운 천간 → 일간과의 십신 관계
        ├─ 세운 지지 → 사주 지지와의 형충회합
        └─ 세운 오행 → 용신/기신과의 관계

Step 4: 대운 + 세운 복합 분석
        └─ 대운과 세운의 상호작용 (상생/상극/충합)

Step 5: 각 운세별 세부 해석
        ├─ 총운: 전체적 방향성
        ├─ 직장운: 관성, 인성, 월주 중심
        ├─ 연애운: 재성/관성, 일지, 도화 중심
        ├─ 재물운: 재성, 식상, 비겁 중심
        └─ 건강운: 오행 균형, 충형, 식신 중심

Step 6: 월별/계절별 운세 세분화
        └─ 용희기신이 강해지는 시기 특정
```

### 4-2. 해석 시 주의사항

```
⚠️ 전문가가 지켜야 할 원칙

1. 용신(用神) 선정의 중요성
   └─ 같은 십신이라도 용신이면 (+), 기신이면 (-)

2. 원국의 격국(格局) 고려
   └─ 종격(從格), 화격(化格) 등은 일반 해석과 다름

3. 대운과의 연계 필수
   └─ 세운만으로 판단 금지, 반드시 대운 흐름 확인

4. 음양/강약 균형 고려
   └─ 단순 오행 개수가 아닌 계절/위치 고려

5. 과도한 길흉 단정 자제
   └─ "반드시 ~한다"보다 "~할 가능성이 높다"
```
